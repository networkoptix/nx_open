License Agreement page

Use this page to enter the text of your end user license agreement (EULA). The link to this page is {{portal url}}/content/eula.

The software license agreement is a contract defining the rights and responsibilities of the purchaser and product provider of a proprietary software product. Default EULA text is predefined but can be edited by contacting Network Optix. You can use the default text with customized variables or change the default text as well.

All fields on this page are required.

Available fields are:

Language – select from the language pulldown menu. See Selecting a language.

EULA text – enter the content of your corporate end user license agreement or use the default text provided, customized with the following variables:

Legal Company Name – enter the legal company name, which can be used as a variable (%LEGAL_COMPANY_NAME%) in the EULA text field.

Legal Company Location –  enter the legal company location which can be used as a variable (%LEGAL_COMPANY_LOCATION%) in the EULA text field.

Legal Company Jurisdiction – enter the legal company jurisdiction which can be used as a variable (%LEGAL_COMPANY_JURIDICTION%) in the EULA text field.

Legal Company Court – enter the legal company court system which can be used as a variable (%LEGAL_COMPANY_COURT%) in the EULA text field.
