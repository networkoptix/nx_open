Cloud Maintenance (503) page

This page is displayed when Cloud Portal service is unavailable. The link to this page is {{portal url}}/static/503.html. For example, https://nxvms.com/static/503.html.

Available fields are:

Content – the message that will show on the Cloud maintenance page (503). This is an HTML field.

Download link – (required). Enter a link for a URL where applications can be downloaded when the Cloud Portal is down.

Service maintenance image – choose a .png format file that will display above the text on the 503 page.

Service maintenance image (high res) – choose a .png format file that will display above the text on the 503 page, for high-resolution monitors. Typically this will be the same image, but larger than the other.
