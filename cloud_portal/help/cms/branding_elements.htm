Branding

Use this page to create and edit the primary look and feel of your Cloud Portal.

Available fields are:

Product Name – enter the name of the Cloud Portal. Contents of this field display as the title on the portal landing page.

Skin – choose the default color for accent text and line graphics on your portal. Note that changing the skin requires a forced update.

VMS Name – enter the name of your VMS product. Contents of this field are displayed as the product name for downloads, e.x. in the download {{VMS_Name}} shortcut footer. Avoid using more than 20 characters.

Cloud tab name – enter a name for the Cloud Portal on the tab of the Desktop client.

Support Link – enter /content/support to use the support the embedded CMS support page, or enter a complete URL if you prefer to use another page such as your company's existing support page (ex. https://support.networkoptix.com/hc/en-us/).

Support Link Text – enter the display text to be shown as the support link in emails.

Privacy Policy link – (required). Enter a complete URL to link to an existing privacy policy page, or enter the short form (ex. /privacy-policy/) to use the privacy policy page embedded in the CMS (https://{{CLOUD_ADDRESS}}/content/privacy page.)

Company web site – enter the full link for your company's main web site.

Company Name – (required). Enter the company name.

Client Protocol – select the URL protocol for calling native clients from a browser. If you are unsure of the proper choice do not change this value. Changing it may negatively affect the interaction between the Cloud Portal and Desktop client

IOS application ID – select an App ID for the iOS mobile application. Do not change this value unless you are sure it is necessary. Changing it can cause issues when interacting with the mobile client.

IOS application link  – select a link for the iOS application in the Apple AppStore.

Android application link – enter a link for the Android application in Google Play.

Mail from - name – (required). Enter a sender name for emails coming from the Cloud Portal.

Mail from - email – (required). Enter a sender email address for emails coming from the Cloud Portal. Note that this entry is case sensitive and must be verified by Amazon. Contact support if you need assistance with this.

Favicon – upload a favicon image for Cloud Portal tabs. Must be .ico format.

Portal header logo – upload a branding logo that will display on every page. Must be .png format and have a height of 64 pixels.

Email logo – upload the logo that will used in outgoing emails. Must be .png format. Width must be 32-240 pixels and height must be 32-64 pixels.
