Landing page

The landing page has a predefined structure which is an image, title (from the Product Name field of the branding page), subtitle, and three centered blocks pf text, each with a header of their own. (These block do not have a visible outline.) The link to this page is {{portal url}}/content/about.

Available fields are:

Language – select from the language pulldown menu. See Selecting a language.

Landing page large image – main image for the front page. This image must have a transparent background and the height cannot be less than 240 pixels. The format must be .png. This image will be used for every language. Check clear value to remove the image entirely.

Subtitle – Text that appears under the product name on the landing page. Displays in one line of 50-100 characters.

Left/Middle/Right block title – enter one to three words as the title for a given block. This text will be centered over the block.

Left/Middle/Right block text –  enter eight to sixteen words as the content for a given block. This text will be centered within the block.
