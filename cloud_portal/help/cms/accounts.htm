Accounts

Use this page to view and export the list of registered users.

This list can be sorted by clicking on the column headers for created date, last login, subscribe, is staff, language and customization. You can use the filter table to the right of the list to filter the user list by subscribe status, staff status, date created or last login date. There is also a search tool that applies to all fields in the list.

The count next to the Search field shows the number of users currently listed and the (total number of users). The count next to the Action field shows the number of users selected from the number of users currently listed.

To download the user list

1.If desired, filter the user list.
2.Select specific users by checking their individual checkbox, or click the checkbox in the header to select all users currently displayed in the list.
3.In the Action field, select Export selected objects as csv file.
4.Click Go to download a .csv with the generic name api_account (x).csv, where x is a number that increments automatically.

IMPORTANT! Note that it is up to Cloud Portal administrations to adhere to the user privacy policies of their company when using downloaded user data.

To view information for a specific user

Click on a user email to open a Change account page that displays the data base values for that user. Attributes shown are:

•First name – the user's first name.
•Last name – the user's last name.
•Is active – Displays a green check if user is active or a red x if user is inactive. Creating an account does not make a user active. To activate an account, a user must reply to an account verification email. This status can only be changed by a user having special permissions or by contacting support.
•Activation date – date and time the account verification email reply was received.
•Subscribe – Displays a green check if user is subscribed to receive Cloud Portal emails or a red x if not. This setting can be changed by the user or by administrators.
•Is staff  –  Displays a green check if user is a company employee or a red x if not. Staff users have special permissions (ex. Content Manager or Portal Manager) that are assigned by Network Optix.
•Last login  – date and time the user last logged in to the Cloud Portal.
•Customization – the version of the Cloud Portal the user is able to view.
•Created date – date and time the user account was created on the Cloud Portal.
•Email – the user's email address.
