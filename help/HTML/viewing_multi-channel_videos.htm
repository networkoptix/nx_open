Viewing Multi-Videos

After performing Multi-Video Export exported file is displayed in local files by the following way:

It is possible to perform the following actions:

•View a single camera from Multi-Video. Perform actions described in "Adding Items to Layouts".

•Open whole Multi-Video. It will be opened as regular Layout on separate Tab. Perform actions described in "Opening and Closing Saved Layouts".

•Delete items from Multi-Video. See "Changing Multi-Video after Export".

When Multi-Video is opened it is possible to do the regular actions:

•Navigation through archive

•Thumbnail Navigation

•Thumbnail Search

•Smart Motion Search

•Calendar Search

•Tours

•Screenshots

•Single Camera Export

•Export (incl. Multi-Video Export)

•Watermark Validation of single videos

•Layout Reconfiguration

•Forensic Analysis.

In other words whole bunch of actions DW Spectrum allows to do with real Cameras can be performed with Multi-Videos.
