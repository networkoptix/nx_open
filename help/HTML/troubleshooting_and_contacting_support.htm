Troubleshooting and Contacting Support

To contact support use the following link:

Enter value

If posting an issue, it is necessary to describe the problem as detailed as possible. It may be useful to provide additional information such as: log files, network configuration, etc.

First information that should be provided is

•Hardware used

•DW Spectrum components and drivers versions

•External Libraries versions.

To obtain this information, open Main Menu and choose About (or press F1). The following form will be displayed:

Then, the information about network configuration can be useful if experiencing problems, which should be provided to Support Team in addition to other details.

This information can be acquired by standard Windows tools (i.e.) ipconfig, but DW Spectrum provides a much more simple and intuitive way.

To obtain network configuration, open Main Menu and choose System Settings. Network configuration is displayed in Network Interfaces section of the General tab:

Finally the Client Log Files may be required. Those are located in the following folder:

$USER\AppData\Local\Network Optix\Network Optix DW Spectrum Client\log
