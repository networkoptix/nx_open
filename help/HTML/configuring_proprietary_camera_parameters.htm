Configuring Proprietary Camera Parameters

DW Spectrum allows configuring basic image settings inside the software. These settings are available for Onvif-compliant cameras only.

To go to proprietary camera settings Open camera settings and select Advanced.

screenshot

All settings are applied immediately. Best practice is to leave a video item on the scene while configuring its parameters.

Other proprietary camera parameters such as can be configured directly from camera’s web page.

1.Open camera settings and select General.

2.Enter authentication parameters if the camera requires authentication (Admin only).

3.Click on Web Page. The browser will open camera’s web page.

Web page may be inaccessible because it may be located in a different network. It is possible to check if a camera is accessible by pressing Ping button before opening a web page.
