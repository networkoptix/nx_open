Taking a Screenshot

Screenshot can be taken both from cameras and local files. Both .png and .jpg are supported as output formats.

To take a screenshot from a video:

1.Select the desired item.

2.Seek to the desired position (frame seek will help).

3.Invoke Context Menu and choose Take Screenshot (or use Alt + S).

4.Choose the desired file name and location and click Save (Cancel will close the dialog and data will not be saved):

It is possible to perform standard files and folders operations in this form (like in Windows Explorer).

After saving the screenshot will be available in local files:

IMPORTANT: this resource will be available until restart of the client. To make it available permanently, save the exported video to DW Spectrum Media Folder (see “Configuring Media Folders”).
