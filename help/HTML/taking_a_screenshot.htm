Taking Screenshot

Screenshot can be taken both from Cameras and Local Video Files. Both PNG and JPG are supported as output formats.

To take Screenshot from a video:

1.Select the desired Item on Scene.

2.Seek to the desired position (frame seek will help). See "Navigating through Archive and Live".

3.Invoke Context Menu and choose Take Screenshot (or use Alt + S).

4.Choose the desired file name and location and click Save (Cancel will close the dialog and data will not be saved):

It is possible to perform standard files and folders operations in this form (like in Windows Explorer).

After saving Screenshot will be available in Local Files within Resource Tree:

IMPORTANT. This resource will be available until restart of Client. To make it available permanently, save the exported video to DW Spectrum Media Folder (see “Configuring Media Folders”) or make Layout containing this video.
