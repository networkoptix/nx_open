Setting up Recording Schedule

To record a camera, it is necessary to set up a recording schedule first.

DW Spectrum provided intuitive and transparent license policy. Оne license records one camera only. See “Obtaining and Activating DW Spectrum Licenses (Admin Only)”. Even if the recording is not performed at the moment, the license is still considered as used.

 

Recording schedule specifies the following recording parameters:

•Recording Type:

•Record Always

•Motion Only – records only when motion occurs. Motion recording may need motion detection to be supported on the corresponding cameras.

•Motion + Low Quality Always – records always in low quality, but if motion occurs recording is switched to high quality. This recording mode requires dual streaming should be supported on the camera. See "Dual Stream Recording (RADASS)".

Motion Mask and Motion Sensitivity may need to be configured as well  (See “Setting up Motion Mask and Motion Sensitivity (see "Setting up Motion Mask and Motion Sensitivity").

•Recording FPS

•Recording Quality

Note that FPS and Quality recording settings will affect live stream viewing. For example, if 1 FPS and low quality is set in the recording schedule, DW Spectrum will stream live as per selected settings even if the camera is capable of more. If recording is not set, DW Spectrum will stream live at maximum FPS and quality.
