Working with Several DW Spectrum Windows

It is possible to set up layouts on several DW Spectrum windows in a multi-monitor environment.

To open a new window, click on Main Menu –> New –> Window (or press Ctrl + N).

Also the selected items may be opened in a new window:

1.Select the desired items in Resource Tree or on Scene (multi-selection can be applied).

2.Right-click on Context Menu and select Open in New Window.

Finally, it is possible to drag items from one window to another (only Admin can drag items to predefined layouts):

3.Select the desired items in Resource Tree or on Scene (multi-selection can be applied).

4.Hold Ctrl and Drag selected items to a new window.
