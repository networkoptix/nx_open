Network Requirements

It is recommended to have two network adapters to run DW Spectrum:

•Public/Intranet: Internet/Intranet access. This NIC can be used to access DW Spectrum from outside. The connection should be protected. Not required if all components are installed on the same machine.

•Private: Camera network. This NIC should be used for DW Spectrum Server to access cameras. This network connection should be completely closed to any outside access.

The following network configurations are possible:

•All-In-One. All DW Spectrum components are installed on the same machine. Public network can be used for Internet surfing, Intranet access or configuring DW Spectrum components from the outside. For standalone systems, public network is not required unless inbound or outbound access for computer is required.

•Client – Server. The components and server are installed on separate computers. The server uses one NIC (private) to acquire data from cameras while the second one communicates with clients.

 

•Enterprise Installation. All system components are installed on different computers. Two NICs are installed on the computer EC is running on (first-one is used by clients to connect to EC from public network and the second-one – to connect to the private camera network). Media servers are located in separate computers in the private network as well and connected to EC via private network address. Traffic from Media Servers is proxied to Public Network by Mediaproxy (see “Forwarding Traffic from Private to Public Subnets via Mediaproxy Component”). In this case Media Server can be installed along with Enterprise Controller.

In this case in order to attach cameras from another private network it is necessary to set up an additional media server (Media Server 3 on the image above). It should have two NICs: one is used to connect to cameras (private), the other is used to connect to EC (public).

Some operations such as camera configurations require access to a camera web page. If DW Spectrum client is installed on a different computer, it may not be able to connect to a private camera network.

In this particular case, it may be required to set up a VPN server on the machine that DW Spectrum Media Server is running on. In order to open a camera web page, a user must first establish a VPN connection with DW Spectrum Media Server, which will allow for the camera network to be accessible.
