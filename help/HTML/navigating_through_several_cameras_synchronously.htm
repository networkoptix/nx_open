Navigating through Several Cameras Synchronously

DW Spectrum provides a powerful engine that enables users to navigate through multiple cameras: if several Cameras are displaying at the same time, they can be synchronized.

If User performs search, fast forward, rewind or search by frame – each camera will be synchronized.

However, recording may be set up differently. If no recording took place over a certain period of time, NO DATA will appear on the camera view.

Recorded fragments are displayed on the Time Line in two rows:

•The upper row displays fragments for Camera selected on Scene

•The lower row – for all Cameras displayed on Scene

It is possible to disable Synchronization (press button). If no Item is selected, the selection will move to the previously selected item, which will become the only item affected by playback controls (seek, speed etc).

Thus, if Synchronization is disabled, it is possible to seek each Camera in a different position.

If User selects Camera item on Scene and clicks again, all other cameras will get synchronized with the selected Camera (Position and Speed will be synchronized).
