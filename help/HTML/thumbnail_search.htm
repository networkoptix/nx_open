Thumbnail Search

This feature can greatly help searching through large archives. It breaks the selected period of time into smaller parts and represent these parts as separate items. For instance it breaks a month period into 10 3-days periods displaying them as separate items. Then it is possible to roughly locate the event in one of these periods and break this period in 9 8-hour periods, etc. Finally location of a specific event may take 3-5 iterations for initial period of several months.

To perform Thumbnail Search:

Select the desired camera on Scene.

Select the region to export on Time Scale:

Use Right Click Drag

Open Context Menu on Time Scale and click Mark Selection Start (or press “[”), then Mark Selection Start (or press “]”)

[PICTURE _img34.emf]

 To clear selection use the corresponding Context Menu item

Adjust the region to export by dragging edges

To make zooming easier, navigate through Timeline using a mouse wheel.

Right-click for Context Menu and choose Thumbnails Search. New tab will be opened:

This tab contains several items representing the selected camera in different periods. If click on items, the selection on Time Line will display the particular period in archive related to the selected item.

If the selection does not contain archive data this feature is not available.

Repeat steps 1-3 on the desired item to refine search.

 

It is possible to perform the following actions in Thumbnail Search tab:

Naviration through archive,

Thumbnail Navigation,

Smart Motion Search,

Calendar Search,

Tours,

Screenshot

export,
