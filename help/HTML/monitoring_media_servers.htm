Monitoring Media Servers (Admin Only)

DW Spectrum enables users to perform monitoring the health of Media Servers in the real-time mode. To do so: perform one of the following:

•Drag Media Server from Resource Tree to Scene

•Open Media Server's Context Menu and choose Monitor (Monitor in a New Tab or Window)

The information is displayed by the following way:

•CPU Load

•Memory Usage (RAM)

•Hard disk partitions usage (C: and D: in this case).

It is possible to do same things as any other item: move, resize, swap, duplicate etc (see "Items on Layouts").

To stop monitoring just remove the corresponding item from Scene.

Several Media Servers can be monitored simultaneously.
