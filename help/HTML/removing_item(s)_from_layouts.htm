Removing Item(s) from Layouts

– If select several items and press "X" all items were closed. Now only one is.

 

IMPORTANT: Viewers cannot remove items from predefined layouts. They can only open new tabs and add/remove items, but cannot save.

To remove item(s) from a layout on Scene:

1.Select desired item(s) on Scene.

2.Proceed with one of the following:

•Press to remove a single item,

•Right-click on Context Menu and select Remove from Layout (or hit Del) to remove all selected items.

3.The following message will appear to confirm deletion.

To remove an item(s) from a layout in Resource Tree:

4.Expand users and locate the desired layout in Resource Tree.

5.Select a desired item(s) under the specified layout. Multi-selection can be applied (see “Working with a Resource Tree”).

6.Open Context Menu and select Remove from Layout (or hit Del).

7.Confirm deletion by clicking OK.
