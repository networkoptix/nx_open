Viewing Videos Exported from Single Camera

As soon as export is finished, the video will be available in Local Files in Resource Tree. Note that AVI and MKV files are  shown as a single record and EXE file - as Folder containing exported Camera:

It is possible to open either EXE file - it will be displayed in new Tab or open a camera - it will be displayed as single Item.

When an exported file is opened, Time Line will display the exported time period:

If EXE bundle is opened the recorded fragments and motion are displayed on the Time Line as well:

Note: start and end time bounds differ from a regular Local File. It is the start and the end time of the exported fragment. If view info the time will be displayed at the bottom right side of the item as well.
