Renaming a Camera

If a camera is discovered automatically (see "Discovering Cameras Automatically") it is displayed in Resource Tree as predefined string ("model"or "manufacturer + model".

It is recommended to rename same model cameras for easier identification. Cameras can be renamed directly in Resource Tree using Context Menu or in camera settings:

IP-addresses in name are non-editable as they are added to a camera name automatically.  It is possible to turn off IP addresses displaying  in Resource Tree.
