Changing Multi-Videos after Export

DW Spectrum provides a lot of possibilities when working with Multi-Videos. User can do almost everything as in regular Layout:

•Add or remove Items. For instance, if Multi-Video was exported since 10PM till 11PM and if a camera is added it will be synchronized with the rest and will be displaying the same archive (10PM – 11PM). See "Adding Items to Layouts" and "Removing Item(s) from Layouts".

•Adjusting Layout Look and Feel (move, swap, resize, rotate). See "Adjusting the Layout Look and Feel".

•Save modified Multi-Video. One restriction is applied: it is not possible to mix Local Videos and Cameras on a single Multi-Video. Same as Layouts (see "Saving Layouts (Admin Only)") but can be performed by Viewers.

Note: Multi-Video can be saved with read-only flag. In this case it is possible to modify it but saving is not allowed.
