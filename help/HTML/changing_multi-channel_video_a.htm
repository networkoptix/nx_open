Changing Multi-Channel Video after Export

DW Spectrum provides a lot of possibilities when working with multi-channel videos. User can do almost everything as in a regular layout:

Add or remove items. For instance, if multi-stream video was exported since 10PM till 11PM and if a camera is added it will be synchronized with the rest and will be displaying the same archive (10PM – 11PM). See "Adding Items to Layouts" and "Removing Item(s) from Layouts".

Adjusting the Layout Look and Feel (move, swap, resize, rotate). See "Adjusting the Layout Look and Feel".

Save modified multi-channel video. One restriction is applied: it is not possible to mix local videos and camera on a single multi-video. Same as layouts (see "Saving Layouts (Admin Only)") but can be performed by viewers.

Note that multi-channel video can be saved with read-only flag. In this case it is possible to modify it but saving is not allowed.
