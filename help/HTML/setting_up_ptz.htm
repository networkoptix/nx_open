Setting up PTZ

DW Spectrum provides smart and intuitive way to setup PTZ on cameras. PTZ can be set by Administrator or by Viewer that has corresponding permissions (see "Users Management").

The feature is available if:

it is supported on a camera (Onvif cameras only).

camera in on Live.

If these requirements are met is displayed on the corresponding camera item.

To set up PTZ press . and buttons are displayed, red arrow is shown over the video.

Use arrow to change point and use and buttons to zoom in/out.

After all settings are configured press once again.
