Deleting Layouts (Admin Only)

To delete Layout from Resource Tree:

1.Find and select desired Layout(s) in Resource Tree.

2.Invoke on Context Menu and choose Delete Layout (or press Del).

3.Accept deletion by clicking on Yes or press Cancel to discard all changes.

If the layout is opened on another computer is deleted, all items will disappear.
