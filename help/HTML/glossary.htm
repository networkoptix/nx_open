Glossary

Administrator – user that can configure DW Spectrum.

Archive – video and audio data recorded from cameras.

Aspect Ratio (AR) – Ratio of Height and Width of a video. Most commonly used are: 4:3 and 16:9.

Bitrate – amount of bytes per second. It is used to measure video stream.

Client – the software used for connecting to servers and view video streams.

Codec – video or audio compression that is used.

Dual Streaming – feature that enables a camera to provide 2 separate streams simultaneously. This may result in bandwidth and processor saving on Client (see RADASS).

Export – ability to grab a piece of archive in order to distribute it or provide as an evidence. Exported video can be viewed on any device.

FPS – Frames per Second. It is used to measure video stream.

DW Spectrum Server Components – DW Spectrum Media Server and Enterprise Controller.

Item – video or image on Scene.

Layout – the saved video items and their position size and orientation. It is used to present surveillance information to DW Spectrum user.

Live – ability to view cameras "as is" in real-time mode.

Media Server Port – the port is used by Media Server to process requests for recorded fragments and video data from DW Spectrum Client.

Motion Detection – feature that indicates if the motion occurred in a camera’s viewing zone.

Motion Mask – the area in viewing zone that will not trigger Motion Detection.

Multi-Video Export – ability to grab several videos in proprietary or bundled format.

OS – Operating System.

Private (cameras) Network – the network used to transmit data from cameras to Media Server. It is not accessible from outside.

Public (Intranet) Network – the network used to connect to Media Server from outside. It can be used either for connecting Media Server to Enterprise Controller or to configure Server Components. It may or may not be connected to the Internet.

PTZ – Point, Tilt, Zoom. If camera supports PTZ it can be controlled.

Resources – cameras, Media Servers, local video files, users and layouts.

RADASS – Resolution and Algorithmic Data  Adaptive Scaling System. Enables to dynamically switch resolution in order to perform bandwidth saving and optimize processor load. Requires Dual-Streaming is supported on Cameras.

RTSP Port – the port is used by Media Server to process requests for media streams from DW Spectrum Client.

Server – the computer DW Spectrum Media server is installed on.

Smart Motion Search – ability to search by motion that occurred in selected region. DW Spectrum will provide fragments with  motion occurred in the region specified.

Super Administrator – initial user that is allowed to do everything in DW Spectrum (his login is admin). It cannot be deleted.

Sync – ability to playback several cameras simultaneously.

Thumbnail Search – the feature breaks the selected period of time into smaller parts and represent these parts as separate videos. For instance it breaks a month period into 10 3-days periods displaying them as separate. This helps users to perform search in large archives.

Thumbnails – the feature draws small previews on top or the recorded fragments making navigation easier.

URL – uniform resource locator is a specific character string that constitutes a reference to an Internet/Intranet resource. It is used to establish connection to Enterprise Controller.

Viewer – user that can do a limited number of actions in DW Spectrum (usually he is not allowed to change configuration).

Watermark – the feature is used to check the validity of the exported files. If a file was changed somehow (even 1 byte of information is changed) the file will be indicated as invalid.
