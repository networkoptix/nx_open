Quick Installation (All-in-One)

Choose this installation type if you want to install all DW Spectrum components on the same computer without specifying any parameters.

To do a quick All-in-One installation:

1.Run the installer.

2.Choose Quick Full as installation type.

3.Set up a password for Enterprise Controller. This password will be used every time DW Spectrum components are connected to Enterprise Controller:

The installation has been completed.

All DW Spectrum components are now installed and the following default parameters will be applied:

•Installation Folder: C:\Program Files (x86)\Network Optix\DW Spectrum\ (may be different in x86 systems)

•Enterprise Controller parameters:

•Port: 7001

•Proxy Port: 7004

•Login: admin

•Password: specified by the user earlier

•Media Server parameters:

•IP Address to access EC: 127.0.0.1 or local host

•Port to access EC: 7001

•Login to access EC: admin

•Password to access EC: specified by the user earlier

•API Port (HTTP Port for Media Server): 7003

•RTSP Port (Port for media traffic): 7002

•Recording Folder: C:\DW Spectrum\

•Client Parameters:

•IP Address to access EC: 127.0.0.1 (or local host)

•Port to access EC: 7001

•Login to access EC: admin

•Password to access EC: specified by the user earlier

Note. If any of the default ports are busy, the installer will automatically choose the next available port (7011, 7012 etc. for EC, 7013, 7014 for Media Server API Port etc.).
