Screen Recording (Windows Only)

This feature is extremely useful for performing presentations in DW Spectrum. It is possible to record from several displays, record sound along with video, choose different quality, etc.

Screen is recorded in the following format:

•Video: MPEG4 Part 2

•Audio: Stereo (Lame)

•Container: AVI

IMPORTANT. It is necessary to have a powerful processor and discrete video adapter to do screen recording. See recommended configuration in the installation guide.
