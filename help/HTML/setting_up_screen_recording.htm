Setting up Screen Recording

To set up Screen Recording:

1.Open Main Menu and choose System Settings.

2.Go to Screen Recording tab.

3.Configure the Screen Recording parameters:

•Select Screen to capture: if dealing with more than one display, it is possible to select which display is to be captured.

•Capture mode: Fullscreen or DW Spectrum Window.

•Disable Aero – choose this option to enhance performance. If chosen, once Screen Recording is in progress, Windows Aero will be turned off.

•Capture Cursor – select this checkbox if mouse cursor needs to be shown during recording.

•Resolution – select the resolution of screen capture. The less the resolution, the higher the performance.

•Encoder Quality – select Performance for best performance select Best for best quality.

•Primary device and Secondary device – select sound devices to grab audio from. Audio will be mixed down from both devices. The best practice is to select master from the sound card as primary and microphone as secondary.
