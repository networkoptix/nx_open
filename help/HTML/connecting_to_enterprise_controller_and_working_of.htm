Connecting to Enterprise Controller and Working Offline

In order to get access to Media Server and cameras, the user needs to be connected to Enterprise Controller.

The connection to Enterprise Controller can be established by pressing the Connect button ( or ) or via Main Menu –> Connect to (Another) Server.

The current connection status is indicated by the Connect button color:

– connected

– disconnected

The connection dialog is displayed by the following way:

It is necessary to enter the following values:

•Host – IP Address or name of the computer EC is installed on (localhost or 127.0.0.1 in case of the All-in-One installation).

•Port – IP Port for access to EC (7001 by default).

•Login and Password to connect to EC. If connecting for the first time, admin should be used as a login. Use the same password as specified during the initial installation.

It is possible to press Test to check connectivity to EC. If test is OK then access is granted. The following may cause connection errors:

•EC is not available:

•specified IP Address is wrong or inaccessible

•specified port is wrong

•EC is stopped

•login and/or password are incorrect

•EC and Client are incompatible with each other (different versions)

It is possible to save and restore connection details:

•If the current connection details should be applied further it is necessary to press Save... and enter a connection name to store them. Next time the details will be available in the drop-down list.

•To delete stored connection details choose the desired records in the drop-down list and press Delete.

•If last connection details has not been saved the software will still use then during next launch (* Last Used Connection *).

Note: Passwords are not stored in saved connection details.

If the client is not connected to Enterprise Controller, a user can only deal with local file names (see “Playing Back Local Files in DW Spectrum”).

If needed to connect to a different EC under a different user, bring up the login window and enter the new EC coordinates or login credentials. If canceled, the current user will still be connected to Enterprise Controller.
