Exporting Layout in Multi-Video Format

DW Spectrum provides a powerful feature to perform Export in proprietary format. Comparing to the standard Export (see "Exporting Recorded Region from Camera") it has many benefits:

•it is possible to export several videos simultaneously (for instance it is possible to export the last hour of recording on 5 cameras). Exported files are saved as regular Layout and can be opened in DW Spectrum. It will be possible to navigate through this exported Layout as usual (see "Navigating through several cameras synchronously"), perform Smart Motion Search etc

•it is possible to export additional information (recorded chunks and motion)

•finally it is possible to prepare a bundled version that can be run and viewed on an absolutely clean Windows systems.

To do Export in proprietary format:

1.Place the desired Items on Scene. It is possible to export either Cameras or Local Video Files. It is not possible to mix them.

2.Select the region to export on Time Scale:

•Use Right Click Drag

•Open Context Menu on Time Scale and click Mark Selection Start (or press “[”), then Mark Selection End (or press “]”)

•To clear selection use the corresponding Context Menu item

•Adjust the region to export by dragging edges.

•To make zooming easier, zoom Time Line using a Mouse Wheel.

3.Invoke Context Menu and choose Export Selected Range:

If the selection contains empty archive on Camera it will be exported and NO DATA will be shown  if view exported video.

IMPORTANT. If large amount of time is selected export may take a long. In this case the corresponding warning is given:

4.Choose the desired file name, format and location and click Save (Cancel will close the dialog and no exported data will be saved):

It is possible to perform standard files and folders operations in this form (like in Windows Explorer).

The following formats are supported:

•NOV – proprietary format. Should be opened by DW Spectrum. Can be changed in the future (see "Changing Multi-Video after Export").

•NOV (Read-Only) – proprietary format. Should be opened by DW Spectrum. Cannot be changed in the future.

•EXE – executable bundle (platform dependent – x86 or x64). Can be opened without DW Spectrum installed on the computer. Can be changed in the future (see "Changing Multi-Video after Export").

•EXE (Read-Only) – proprietary format  (platform dependent – x86 or x64). Executable bundle. Can be opened without DW Spectrum installed on the computer.  Should be opened by DW Spectrum. Cannot be changed in the future.

EXE format is used for distributing videos across the audience that may not have any codecs or media players installed. As soon as this bundle is started the Client instance is run and it plays back the exported video. In this case data about motion and recorded fragments is also exported.

IMPORTANT. x64 executable files must be run on the same architecture. For instance, files produced with x64 version will be only viewable on x64 machines, however x86 can be viewed everywhere.

As soon as export is finished, Multi-Video will be available in local files:

It is possible to do the following operations:

•View this videos (see "Viewing Multi-Videos") as Layouts

•View single Items from these videos

•Edit Multi-Videos if not read-only (see "Changing Multi-Video after Export").
