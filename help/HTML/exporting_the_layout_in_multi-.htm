Exporting the Layout in Multi-Video Format

DW Spectrum provides a powerful feature to perform export in proprietary format. Compating to the standard export (see "UNREGISTERED EVALUATION VERSION") it has many benefits:

– it is possible to export several files simultaneously (for instance it is possible to export the lash hour of recording on 5 cameras). Exported files are saver as a regular layout and can be opened in DW Spectrum. It will be possible to navigate through this exported layout as usual (see "Navigating through several cameras synchronously"), perform motion.

– it is possible to export additional information (recorded chunks and motion)

– finally it is possible to prepare a bundled version that can be run and viewed on an absolutely clean Windows systems.

To do export in proprietary format:

 

Add one or several desired cameras on Scene (see "Adding Items to Layouts"). It is possible to use either local files or cameras but it is not allowed to mix them.

Select the region to export on Time Scale:

Use Right Click Drag

Open Context Menu on Time Scale and click Mark Selection Start (or press “[”), then Mark Selection Start (or press “]”)

[PICTURE _img34.emf]

 To clear selection use the corresponding Context Menu item

Adjust the region to export by dragging edges

To make zooming easier, navigate through Timeline using a mouse wheel.

Right-click for Context Menu and choose Export Selected Range:

[PICTURE _img64.bmp]

If the selection does not contain archive data this feature is not available.

Choose the desired file name, format and location and click Save (Cancel will close the dialog and any exported data will not be saved):

[PICTURE _img65.bmp]

It is possible to perform standard files and folders operations in this form (like in Windows Explorer).

*.nov – proprietary format. Should be opened by DW Spectrum. Can be changed in the future (see "Changing Multi-Channel Video after Export").

*.nov (read only) – proprietary format. Should be opened by DW Spectrum. Cannot be changed in the future.

*.exe – executable bundle (platform dependent – x86 or x64). Can be opened without DW Spectrum installed on the computer. Can be changed in the future (see "Changing Multi-Channel Video after Export").

*.exe (read only) – proprietary format  (platform dependent – x86 or x64). Executable bundle. Can be opened without DW Spectrum installed on the computer.  Should be opened by DW Spectrum. Cannot be changed in the future.

IMPORTANT: x64 executable files must be run on the same architecture. For instance, files produced with x64 version will be only viewable on x64 machines, however x86 can be viewed everywhere.

As soon as export is finished, the multi-channel video will be available in local files. It is possible to view this video (see "UNREGISTERED EVALUATION VERSION") and edit it if not read-only (see "Changing Multi-Channel Video after Export").
