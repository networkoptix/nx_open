Setting Up Cameras

It is possible to configure Cameras for Administrators and Viewers that have the appropriate access Permission. Administrators can configure all cameras and Viewers can only use the ones that are assigned to them by Administrators.

A user can locate Camera Settings from Camera's Context Menu.
