Adding Cameras Manually

If a camera is not accessible via broadcast, it cannot be found automaticelly. In this case DW Spectrum provides the ability to add it manually. For instance, it can be useful if a camera is located in a different network or even it can be accessed via Internet only. It is possible to add several cameras simultaneously,

 

The following camera models that can be added manually:

Arecont Vision,

D-Link,

ISD,

All ONVIF-compliant cameras.

 

To add one of several cameras:

Right click on a server and open Context Menu

Choose Add camera(s)... The following form will be displayed:

To add a single camera enter a camera's IP

To add several cameras select Subnet Scan:

Enter the desired IP range to be scanned. By default addresses 0-255 of the same subnet are suggested. This means the whole specified network will be scanned for cameras.

Specify authentication parameters in Login and Password fields (if needed)

Specify Discovery Port if needed. Most cameras are discovered on port 80, so it is better to leave Auto.

Press Scan. The search will be attempted. If cameras are found they are displayed in the list:

 

Select desired cameras and click Add selected.
