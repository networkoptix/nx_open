Launching and Closing DW Spectrum Client

Different ways to launch the application:

•From desktop: DW Spectrum shortcut icon.

•From Start Menu: Start –> Programs –> Digital Watchdog –> DW Spectrum.

•Directly from the installed folder:

•For x86 default is C:\Program Files(x86)\Digital Watchdog\DW Spectrum\Client\DW Spectrum.exe

•For x64 default is C:\Program Files\Digital Watchdog\DW Spectrum\Client\DW Spectrum.exe

To close DW Spectrum client:

•Click on the "" button located in the top right corner.

•Go to Main Menu –> Exit (or press Alt+F4).
