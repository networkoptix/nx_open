Performing Screen Recording

To do Screen Recording:

1.Open Main Menu and choose Start Screen Recording (or press Alt + R).

2.Invoke Context Menu on the Scene (on empty space) and choose Start Screen Recording

3.Click on the icon

4.Screen recording will start in 3 seconds.

5.When finished, do one of the following:

•Open Main Menu and choose Stop Screen Recording (or press Alt + R).

•Invoke Context Menu on the Scene (on empty space) and choose Stop Screen Recording

6.5. Choose the desired file name and location and click Save (Cancel will close the dialog and data will not be saved):

It is possible to perform standard files and folders operations in this form (like in Windows Explorer).

As soon as file is saved it will be available in local files:

IMPORTANT: this resource will be available until restart of the client. To make it available permanently, save the exported video to DW Spectrum Media Folder (see “Configuring Media Folders”).
