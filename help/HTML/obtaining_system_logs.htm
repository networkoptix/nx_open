Obtaining System Logs

It is useful to always keep logs for all DW Spectrum components. Additionally, you can make a simple script for logs archiving and rotation.

DW Spectrum logs are stored in the following paths:

Client: ($USER\AppData\Local\Network Optix\Network Optix DW Spectrum Client\log)

Enterprise Controller and Media Server: to obtain log file locations do the following:

1.Open Tray Tool (see “Configuring DW Spectrum Components after Installation”) and select Show Network Optix Media Server Log or Show Enterprise Controller Log (The selected log file will be opened in Windows Notepad):

2.Choose File –> Save as… in Windows Notepad. Current location if the log file will be displayed there:

It is possible to manually copy text from Notepad and paste id to file.
