通用事件生成器

通用事件是一種讓第三方系統觸發 VMS 動作的特殊事件。 要建立 Generic Event 來整合系統需要向伺服器發出HTTP請求（使用Server API）。 Actions for Generic Events can be configured them using Events&Rules dialog in desktop client application.

Generic Events Generator

Server API

Server API provides a standard set of API calls for 3rd party systems. It gives developers the ability to access almost every system feature available in the regular client: query/manage system resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

API Testing Tool (new) - interactive documentation with tools for testing API calls
API changelog - list of breaking changes in API
API documentation

Camera Details Panel

Special panel for a webclient which displays additional information for cameras:

  • Camera ID
  • Links for all available video-streams
  • Direct export video from browser

To open Camera Details Panel - open web client, select camera, settings icon (gear), then choose "Camera Details"

open Camera Details Panel

影像來源整合 SDK

影像來源整合 SDK 提供整合幾乎任何即時影像或錄影裝置的能力 (IP攝影機, NVR, DVR等等.) SDK 使得搜尋, 顯示, 分析以及錄存影像變得可能. 也支援整合設備的 I/O 埠和位移偵測資訊.

取得 SDK

儲存裝置 SDK

儲存裝置 SDK 提供整合儲存裝置進入系統. 使開發人員可讀取或寫入任何儲存裝置包括本地, 遠端或甚至雲端. 要建立一個儲存裝置插件必須可執行標準功能如：I/O串流, 檢查檔案是否存在, 刪除檔案, 列出資料夾所有檔案等等. 儲存裝置 SDK 還包含如何使 FTP 伺服器成為儲存裝置的範例.
