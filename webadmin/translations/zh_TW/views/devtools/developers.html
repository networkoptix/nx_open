通用事件生成器

通用事件是將事件從第三方系統發送到系統的方法， 可用於在事件規則引擎中觸發系統動作。 通用事件生成器是一個可幫助開發和整合人員的構建工具， 並可測試建立HTTP通用事件。

通用事件生成器

伺服器 API

REST API 能讓開發人員擁有幾乎所有系統功能的權限， 客戶端：查詢/管理系統資源（伺服器，攝影機，使用者，權限), 從系統中索取即時影像或錄影資料, 生成事件，建立規則，操控PTZ攝影機以及更多功能。

API測試工具（新） - API測試互動式文件和工具
API變更日誌 - 上次更新後的API重大更新列表
API文檔 - 可用伺服器API的完整列表

攝影機詳細資訊面板

在 Web 管理介面的瀏覽分頁上顯示對各個攝影機的友善附加開發人員訊息.

  • 攝影機 ID
  • 所有可用的影像流的連結
  • 從瀏覽器直接匯出影像

要開啟攝影機詳細訊息面板 - 到 Web 管理界面, 選擇 瀏覽, 選擇攝影機, 點擊設定圖示(齒輪), 然後選擇“攝影機詳細訊息”

開啟攝影機詳細資訊面板

影像來源SDK

影像來源SDK提供了能把幾乎任何即時或錄製的影像來源（網路攝影機， NVRs，DVRs等）整合到系統中的功能，並可用於發現、顯示、分析 和錄影。 還允許整合I/O 埠和設備，以及從第三方設備擷取位移偵測 信息。

取得 SDK

儲存裝置 SDK

儲存裝置SDK能使開發人員讀取或寫入任何儲存裝置包括本地, 遠端或甚至雲端. 建立儲存插件需要實行標準功能，例如：I / O串流，如果檔案 存在，刪除檔案，資料夾中的檔案清單等。 儲存裝置 SDK包含使用FTP 伺服器作為存儲位置的範例。
