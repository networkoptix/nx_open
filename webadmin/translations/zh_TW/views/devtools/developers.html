通用事件生成器

通用事件是將事件從第三方系統發送到系統的方法， 可用於在事件規則引擎中觸發系統動作。 通用事件生成器是一個幫助建立HTTP通用事件的工具。

通用事件生成器

伺服器 API

伺服器 API 提供一套標準的 API 供第三方廠商呼叫. 讓開發者可取用所有在 Client 上能操作的系統功能: 調閱/管理系統資源 (伺服器, 攝影機, 使用者, 權限), 從系統中索取即時影像或錄影資料, 生成事件, 建立規則, 操控 PTZ 攝影機以及更多功能.

API Testing Tool (new) - interactive documentation with tools for testing API calls
API Changelog - list of breaking changes in API
API Documentation
Server Settings Documentation

攝影機詳細資訊面板

顯示附加開發人員 - 在 Web 管理界面的攝影機友善訊息.

  • 攝影機 ID
  • 所有可用的影像流的連結
  • 從瀏覽器直接匯出影像

要開啟攝影機詳細訊息面板 - 到 Web 管理界面, 選擇 瀏覽, 選擇攝影機, 點擊設定圖示(齒輪), 然後選擇“攝影機詳細訊息”

開啟攝影機詳細資訊面板

影像來源整合 SDK

影像來源整合 SDK 提供整合幾乎任何即時影像或錄影裝置的能力 (IP攝影機, NVR, DVR等等.) SDK 使得搜尋, 顯示, 分析以及錄存影像變得可能. 也支援整合設備的 I/O 埠和位移偵測資訊.

取得 SDK

儲存裝置 SDK

儲存裝置 SDK 提供整合儲存裝置進入系統. 使開發人員可讀取或寫入任何儲存裝置包括本地, 遠端或甚至雲端. 要建立一個儲存裝置插件必須可執行標準功能如：I/O串流, 檢查檔案是否存在, 刪除檔案, 列出資料夾所有檔案等等. 儲存裝置 SDK 還包含如何使 FTP 伺服器成為儲存裝置的範例.
