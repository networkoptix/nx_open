카메라 세부 정보

Web Admin 인터페이스의 카메라에 대한 개발자에게 친숙한 추가 정보를 표시합니다.

  • 카메라 ID
  • 사용가능한 모든 비디오 스트림 주소
  • 브라우저에서 직접 동영상 내보내기

카메라 세부 정보 패널을 열려면 Web Admin 인터페이스로 이동하고 보기를 선택하십시오 , 대상 카메라를 선택하고 설정 아이콘 (장비) 을 클릭 한 다음 "카메라 세부 정보"를 선택하십시오.

영상관련 통합SDK

영상관련 통합SDK는 시스템내 모든 라이브 또는 녹화영상정보(IP 카메라, NVR, DVR, 등...) 에 대하여 가상통합 하는 기능을 제공 합니다. SDK는 검색, 표시, 검출 및 영상녹화에 대한 방법을 생성할수 있습니다. I/O장치 및 모션 검출정보를 연동하는데 사용가능합니다.

저장소 관련 SDK

저장소 SDK는 시스템에 잠재적인 스토리지를 통합 하는 기능을 제공 합니다. 이것을 통하여 개발자들은 모든저장소에 대하여 (내외부, 클라우드) 읽기 또는 쓰기를 수행할수 있습니다. 저장 장치 플러그인을 만들려면 I / O 스트림 (파일이있는 경우), 파일 삭제, 폴더의 파일 목록 등과 같은 표준 기능을 구현해야합니다. 저장소 SDK에는 FTP 서버를 저장 위치로 사용하기위한 예제가 포함되어 있습니다.
