서버 API

서버 API는 타 SW를 위한 기본 API 호출 정보를 제공합니다. It gives developers the ability to access almost every system feature available in the regular client: query/manage system resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

비디오 소스 통합 SDK

비디오 소스 통합 SDK 시스템으로 거의 모든 라이브 또는 녹화 비디오 소스 (IP 카메라, Nvr, Dvr, 등) 을 통합 하는 기능을 제공 합니다. SDK는 검색, 표시, 검출 및 영상 재생 방법을 만듭니다. I/O장치 및 모션 검출정보를 연동하는데 사용가능합니다.

저장소 SDK

Storage SDK는 시스템에 잠재적인 스토리지를 통합 하는 기능을 제공 합니다. 그것은 개발자에서 읽기 또는 쓰기 모든 저장소 위치에 있습니다: 하나는 로컬, 원격, 및 심지어 클라우드. Creating a storage plugin requires implementing standard functions such as: I/O stream, if file exist, delete file, list of files in the folder, etc.Storage SDK also contains an example for using an FTP server as a storage location.
