กิจกรรมทั่วไป

เหตุการณ์ทั่วไปคือวิธีการส่งกิจกรรมจากระบบของบุคคลที่สามไปยังระบบซึ่งสามารถใช้เพื่อเรียกใช้การทำงานของระบบในเครื่องมือเหตุการณ์และกฎ Generic Event Generator เป็นเครื่องมือที่ช่วยให้นักพัฒนาและผู้รวมระบบสร้าง และทดสอบ HTTP Generic Event call

เซิร์ฟเวอร์ API

ช่วยให้นักพัฒนาสามารถเข้าถึงคุณลักษณะของระบบได้เกือบทุกอย่างที่มีอยู่ในไคลเอ็นต์ปกติ: ค้นหา / จัดการทรัพยากรระบบ (เซิร์ฟเวอร์กล้องผู้ใช้สิทธิ์) ดึงวิดีโอสดหรือบันทึกจากระบบสร้างเหตุการณ์สร้างกฎควบคุมกล้อง PTZ, และอื่น ๆ.

เครื่องมือทดสอบ API (ใหม่) - เอกสารแบบโต้ตอบกับเครื่องมือสำหรับการทดสอบ API การโทร
การเปลี่ยนแปลง API - รายการการเปลี่ยนแปลงใน API

รายละเอียดกล้อง

แสดงข้อมูลที่เป็นมิตรกับผู้พัฒนาเพิ่มเติมสำหรับกล้องแต่ละตัวในแท็บมุมมองของ Web Admin Interface:

  • รหัสกล้อง
  • ลิงค์สำหรับวิดีโอสตรีมทั้งหมด
  • ส่งออกวิดีโอโดยตรงจากเบราเซอร์

การเปิดแผงปุ่มรายละเอียดกล้อง - ไปยังอินเทอร์เฟซสำหรับผู้ดูแลระบบเว็บ เลือกมุมมอง เลือกกล้องเป้าหมาย คลิกที่ไอคอนการตั้งค่า (รูปเฟือง), จาก นั้นเลือกกล้องรายละเอียด"

SDK แหล่งวิดีโอ

Video Source SDK ให้ความสามารถในการรวมแหล่งวิดีโอสดหรือที่บันทึกไว้เกือบทั้งหมด (กล้อง IP, NVR, DVRs, ฯลฯ) เข้าสู่ระบบและทำให้สามารถสร้างวิธีการค้นพบแสดงวิเคราะห์ และบันทึกวิดีโอ นอกจากนี้ยังอนุญาตให้รวมพอร์ต I / O และอุปกรณ์รวมทั้งดึงข้อมูลการตรวจจับการเคลื่อนไหว จากอุปกรณ์ของบุคคลที่สาม

SDK ที่เก็บข้อมูล

Storage SDK ช่วยให้นักพัฒนาสามารถอ่านหรือเขียนข้อมูลจากที่เก็บข้อมูลใดก็ได้: โลคัลรีโมทและแม้แต่คลาวด์ เดียว การสร้างปลั๊กอินหน่วยเก็บข้อมูลจำเป็นต้องใช้ฟังก์ชั่นมาตรฐานเช่น: I / O สตรีมหากมีไฟล์ อยู่ให้ลบไฟล์รายชื่อไฟล์ในโฟลเดอร์ ฯลฯ Storage SDK มีตัวอย่างสำหรับการใช้เซิร์ฟเวอร์ FTP เป็นที่เก็บข้อมูล
