A Generic Event is a method of sending events from 3rd party systems to a System, which can be used to trigger System actions in the Events & Rules engine. The Generic Event Generator is a tool which helps build HTTP Generic Event calls.

เซิร์ฟเวอร์ API

API ของเซิร์ฟเวอร์มีชุด API API มาตรฐานสำหรับระบบอื่นๆ ช่วยให้นักพัฒนาสามารถเข้าถึงคุณลักษณะของระบบได้เกือบทุกอย่างที่มีอยู่ในไคลเอ็นต์ปกติ: ค้นหา / จัดการทรัพยากรระบบ (เซิร์ฟเวอร์กล้องผู้ใช้สิทธิ์) ดึงวิดีโอสดหรือบันทึกจากระบบสร้างเหตุการณ์สร้างกฎควบคุมกล้อง PTZ , และอื่น ๆ.

รายละเอียดกล้อง

Displays additional developer-friendly information for cameras in the Web Admin interface:

  • รหัสกล้อง
  • ลิงค์สำหรับวิดีโอสตรีมทั้งหมด
  • ส่งออกวิดีโอโดยตรงจากเบราเซอร์

To open the Camera Details Panel - navigate to the Web Admin interface, choose View, select the target camera, click on the settings icon (gear), then choose "Camera Details"

รายละเอียดกล้อง

SDK การรวมแหล่งที่มาของวิดีโอ

ซอร์สโค้ด SDK ของวิดีโอนำเสนอความสามารถในการรวมแหล่งวิดีโอสดหรือที่บันทึกไว้ (กล้อง IP, NVRs, DVRs ฯลฯ ) เข้ากับระบบ SDK ช่วยในการสร้างวิธีการในการค้นหาแสดงการวิเคราะห์และบันทึกวิดีโอ นอกจากนี้ยังช่วยให้สามารถรวมพอร์ต I / O อุปกรณ์และข้อมูลการตรวจจับการเคลื่อนไหว

SDK ที่เก็บข้อมูล

Storage SDK ช่วยให้สามารถรวมระบบจัดเก็บข้อมูลที่มีศักยภาพลงในระบบได้ ช่วยให้นักพัฒนาสามารถอ่านหรือเขียนข้อมูลไปยังที่เก็บสินค้าได้ทั้งแบบท้องถิ่น, ระยะไกลและแบบคลาวด์ การสร้างปลั๊กอินที่เก็บข้อมูลจำเป็นต้องใช้ฟังก์ชันมาตรฐานเช่นสตรีม I / O หากไฟล์มีอยู่ลบไฟล์รายการไฟล์ในโฟลเดอร์เป็นต้น SDK Storage ยังมีตัวอย่างสำหรับการใช้เซิร์ฟเวอร์ FTP เป็นที่เก็บข้อมูล
