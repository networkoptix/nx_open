เซิร์ฟเวอร์ API

API ของเซิร์ฟเวอร์มีชุด API API มาตรฐานสำหรับระบบอื่นๆ ช่วยให้นักพัฒนาสามารถเข้าถึงคุณลักษณะของระบบได้เกือบทุกอย่างที่มีอยู่ในไคลเอ็นต์ปกติ: ค้นหา / จัดการทรัพยากรระบบ (เซิร์ฟเวอร์กล้องผู้ใช้สิทธิ์) ดึงวิดีโอสดหรือบันทึกจากระบบสร้างเหตุการณ์สร้างกฎควบคุมกล้อง PTZ, และอื่น ๆ.

เครื่องมือทดสอบ API (ใหม่) - เอกสารแบบโต้ตอบกับเครื่องมือสำหรับการทดสอบ API การโทร

รายละเอียดกล้อง

แสดงข้อมูลผู้พัฒนามิตรเพิ่มเติมสำหรับกล้องในอินเทอร์เฟซเว็บ Admin:

  • รหัสกล้อง
  • ลิงค์สำหรับวิดีโอสตรีมทั้งหมด
  • ส่งออกวิดีโอโดยตรงจากเบราเซอร์

การเปิดแผงปุ่มรายละเอียดกล้อง - ไปยังอินเทอร์เฟซสำหรับผู้ดูแลระบบเว็บ เลือกมุมมอง เลือกกล้องเป้าหมาย คลิกที่ไอคอนการตั้งค่า (รูปเฟือง), จาก นั้นเลือกกล้องรายละเอียด"

รายละเอียดกล้อง

SDK การรวมแหล่งที่มาของวิดีโอ

ซอร์สโค้ด SDK ของวิดีโอ นำเสนอความสามารถในการรวมแหล่งวิดีโอสดหรือที่บันทึกไว้ (กล้อง IP, NVRs, DVRs ฯลฯ) เข้ากับระบบ SDK ช่วยในการสร้างวิธีการในการค้นหาแสดงการวิเคราะห์และบันทึกวิดีโอ นอกจากนี้ยังช่วยให้สามารถรวมพอร์ต I / O อุปกรณ์และข้อมูลการตรวจจับการเคลื่อนไหว

SDK ที่เก็บข้อมูล

Storage SDK ช่วยให้สามารถรวมระบบจัดเก็บข้อมูลที่มีศักยภาพลงในระบบได้ ช่วยให้นักพัฒนาสามารถอ่านหรือเขียนข้อมูลไปยังที่เก็บสินค้าได้ทั้งแบบท้องถิ่น, ระยะไกลและแบบคลาวด์ การสร้างปลั๊กอินที่เก็บข้อมูลจำเป็นต้องใช้ฟังก์ชันมาตรฐานเช่นสตรีม I / O หากไฟล์มีอยู่ลบไฟล์รายการไฟล์ในโฟลเดอร์เป็นต้น SDK Storage ยังมีตัวอย่างสำหรับการใช้เซิร์ฟเวอร์ FTP เป็นที่เก็บข้อมูล
