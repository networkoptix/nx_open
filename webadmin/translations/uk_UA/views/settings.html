Потрібно підключити його до мережі або налаштувати правильну IP-адресу.

Порт

Скидання сервера

Ця дія поверне цей сервер до налаштувань за замовчуванням (очистить базу даних і скине конфігурацію). The server will be detached from the current System. Відеоархів залишиться незмінним.

Detach server from the System

This action will detach server from the System keeping database, local users and configuration. Відеоархів залишиться незмінним. Хмарні користувачі будуть видалені.

Перезапуск

Змінити пароль адміністратора

Якщо система локальна (не підключена до хмари), це змінить пароль для користувача "admin" та пароль відновлення (користувач "root") на деяких серверах.
В іншому випадку це змінить лише пароль відновлення.
