Генератор довільних подій

Довільний подія - спосіб відправки подій із сторонніх систем. Довільні події можуть використовуватися для запуску системних дій в механізмі подій і правил. Загальний генератор подій - це інструмент, який допомагає розробникам та інтеграторам              Створюйте та перевіряйте виклики подій HTTP.

API сервера

Це дає можливість розробникам отримувати доступ до майже кожної функції системи, доступної в звичайному клієнті: запиту і управління системними ресурсами (сервери, камери, користувачі, повноваження), отримання відео в реальному часі або архіву з системи, генеріраціі подій, створення правил, управління PTZ -Камера і багато чому іншому.

Інструмент тестування API (новий)              - інтерактивна документація та інструменти для тестування викликів API
             Журнал змін API              - Список змін API з часу останнього випуску
             Документація API              - повний список доступних викликів API сервера

Панель відомостей про камеру

Відображає додаткову інформацію, доступну для розробників, для окремих камер на вкладці "Вигляд" інтерфейсу Webadmin:

  • Ідентифікатор камери
  • Посилання для всіх доступних відеопотоків
  • Прямий експорт відео з браузера

Щоб відкрити панель відомостей про камери - перейдіть в інтерфейс веб-адміністратора, виберіть Вид, виберіть бажану камеру, клацніть значок налаштувань (шестерня), потім виберіть "Відомості про камеру"

SDK джерела відео

SDK відео джерела надають можливість інтегрувати практично будь-який живий або записаний поток відео (IP-камери, відеореєстратори, відеореєстратори тощо) в систему і дозволяє створювати способи захоплення, перегляду, аналізу та запису відео. Він також дозволяє інтегрувати порти та пристрої вводу-виводу, а також знімати інформацію про виявлення руху з пристроїв третьої сторони.

SDK для інтеграції сховищ

SDK зберігання дозволяє розробникам читати або писати з будь-якого місця, локально, віддалено і навіть у хмарі. Створення модуля пам'яті вимагає реалізації стандартних функцій, таких як: потік вводу-виводу, якщо файл існує, видаляти файл, список файлів у папці тощо. SDK зберігання містить приклад використання сервера FTP як місця зберігання.
