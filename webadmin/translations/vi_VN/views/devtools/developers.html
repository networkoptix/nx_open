API máy chủ

API máy chủ cung cấp tập hợp các chuẩn API cho các hệ thống của bên thứ ba. Nó cung cấp cho các nhà phát triển khả năng truy cập hầu hết các tính năng hệ thống có sẵn trong máy tính để: truy vấn / quản lý các tài nguyên hệ thống (máy chủ, camera, người dùng, quyền hạn), kéo video trực tiếp hoặc ghi ra khỏi hệ thống, tạo các sự kiện, tạo quy tắc, điều khiển PTZ camera, và hơn thế nữa.

Thông tin về camera

Displays additional developer-friendly information for cameras in the Web Admin interface:

  • Tên Camera
  • Liên kết cho tất cả các dòng video có sẵn
  • Trực tiếp xuất video từ trình duyệt

To open the Camera Details Panel - navigate to the Web Admin interface, choose View, select the target camera, click on the settings icon (gear), then choose "Camera Details"

Nguồn video tích hợp SDK

Tích hợp nguồn video SDK cung cấp khả năng tích hợp hầu hết các nguồn video trực tiếp hoặc ghi lại (Máy quay IP, NVR, DVR, v. v...) vào Hệ thống. SDK tạo ra các phương pháp để phát hiện, hiển thị, phân tích và ghi video. Nó cũng cho phép tích hợp cổng I/O của thiết bị và thông tin phát hiện chuyển động.

SDK lưu trữ

Bộ nhớ SDK cung cấp khả năng tích hợp lưu trữ tiềm năng vào Hệ thống. Nó cho phép các nhà phát triển đọc hoặc ghi vào bất kỳ vị trí lưu trữ nào: địa phương, từ xa và thậm chí cả dữ liệu đám mây. Tạo một plugin lưu trữ đòi hỏi phải thực hiện các chức năng tiêu chuẩn như: luồng I/O, nếu tệp tồn tại, xóa tệp, danh sách tệp trong thư mục, etc.Storage SDK cũng chứa một ví dụ để sử dụng máy chủ FTP làm vị trí lưu trữ.
