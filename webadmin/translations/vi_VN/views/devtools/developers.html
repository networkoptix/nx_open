A Generic Event is a method of sending events from 3rd party Systems to a System, which can be used to trigger System actions in the Events & Rules engine. Generator Event Generator là một công cụ giúp thiết lập và gọi các hàm HTTP Generic Event.

Máy chủ API

Server API provides a standard set of API calls for 3rd party Systems. It gives developers the ability to access almost every System feature available in the regular client: query/manage System resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

Bảng thông tin về camera

Hiển thị thông tin bổ sung dành cho nhà phát triển sử dụng giao diện quản trị web:

  • Tên Camera
  • Liên kết cho tất cả các dòng video có sẵn
  • Trực tiếp xuất video từ trình duyệt

Để mở Bảng chi tiết cài đặt Camera - hãy sử dụng giao diện Web Admin, chọn View, chọn camera đích, nhấp chuột vào biểu tượng cài đặt (bánh răng), sau đó chọn "Chi tiết camera"

Nguồn video tích hợp SDK

Tích hợp nguồn video SDK cung cấp khả năng tích hợp hầu hết các nguồn video trực tiếp hoặc ghi lại (Máy quay IP, NVR, DVR, v. v...) vào Hệ thống. SDK tạo ra các phương pháp để phát hiện, hiển thị, phân tích và ghi video. Nó cũng cho phép tích hợp cổng I/O của thiết bị và thông tin phát hiện chuyển động.

SDK lưu trữ

Bộ nhớ SDK cung cấp khả năng tích hợp lưu trữ tiềm năng vào Hệ thống. Nó cho phép các nhà phát triển đọc hoặc ghi vào bất kỳ vị trí lưu trữ nào: địa phương, từ xa và thậm chí cả dữ liệu đám mây. Tạo một plugin lưu trữ đòi hỏi phải thực hiện các chức năng tiêu chuẩn như: luồng I/O, nếu tệp tồn tại, xóa tệp, danh sách tệp trong thư mục, etc.Storage SDK cũng chứa một ví dụ để sử dụng máy chủ FTP làm vị trí lưu trữ.
