Tài liệu máy chủ

Các cài đặt này có thể truy cập tại:

Đối với Windows trong khóa đăng ký: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Đối với tất cả các nền tảng khác, thông tin được đặt trong một tệp cấu hình: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. To temporarily comment out some setting, add the underscore symbol ("_") to the begining of its name, e.g. "absoluteRtcpTimestamps=false" converts to "_absoluteRtcpTimestamps=false".
