Generic Events Generator

Generic Events is a special kind of events for 3rd party systems to trigger actions in the VMS. To create Generic Event integrated system needs to make HTTP request to the Server (using Server API). Actions for Generic Events can be configured using Events Rules dialog in desktop client application.

Servidor API

Servidor API proporciona un conjunto estándar de API llamadas para 3 sistemas de partido. Da a los desarrolladores la capacidad para acceder a casi cualquier función de sistema disponible en el cliente regular: consulta y administrar los recursos del sistema (servidores, cámaras, usuarios, permisos), tirar de vivo o video grabado fuera del sistema, generar eventos, crear normas, controlar cámaras PTZ y mucho más.

Camera Details Panel

Special panel for a webclient which displays additional information for cameras:

  • ID de cámara
  • Links for all available video-streams
  • Direct export video from browser

To open Camera Details Panel - open web client, select camera, settings icon (gear), then choose "Camera Details"

Fuente de vídeo integración SDK

Video fuente Integration SDK ofrece la posibilidad de integrar sistema de prácticamente cualquier fuente de vídeo grabado o en directo (cámaras IP, NVR, DVR, etcetera). SDK permite crear métodos para descubrir, visualizar, analizar y grabar video. También permite la integración de puertos de E/S de dispositivo y la información de detección de movimiento.

Almacenamiento SDK

Almacenamiento SDK ofrece la posibilidad de integrar sistema de almacenamiento posibles. Permite a los desarrolladores leer de o escribir en cualquier lugar de almacenamiento de información: nube local, remota e incluso uno. La creación de un plugin de almacenamiento requiere implementar funciones estándar tales como: stream de I/O, si existe archivo, Borrar archivo, lista de archivos en la carpeta, etcetera. Almacenamiento SDK contiene un ejemplo de uso de un servidor FTP como una ubicación de almacenamiento.
