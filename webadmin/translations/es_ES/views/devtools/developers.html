Generador de eventos genéricos

A Generic Event is a method of sending events from 3rd party systems to a System, which can be used to trigger System actions in the Events & Rules engine. The Generic Event Generator is a tool which helps build HTTP Generic Event calls.

Servidor API

Servidor API proporciona un conjunto estándar de llamadas API para sistemas de terceros. Da a los desarrolladores la capacidad para acceder a casi cualquier función de sistema disponible en el cliente regular: consulta y administrar los recursos del sistema (servidores, cámaras, usuarios, permisos), tirar de vivo o video grabado fuera del sistema, generar eventos, crear normas, controlar cámaras PTZ y mucho más.

Panel de detalles de la cámara

Displays additional developer-friendly information for cameras in the Web Admin interface:

  • ID de cámara
  • Enlaces para todos flujos de vídeo disponibles
  • Exportar vídeo directamente desde el navegador

To open the Camera Details Panel - navigate to the Web Admin interface, choose View, select the target camera, click on the settings icon (gear), then choose "Camera Details"

Fuente de vídeo integración SDK

Video fuente Integration SDK ofrece la posibilidad de integrar sistema de prácticamente cualquier fuente de vídeo grabado o en directo (cámaras IP, NVR, DVR, etcetera). SDK permite crear métodos para descubrir, visualizar, analizar y grabar video. También permite la integración de puertos de E/S de dispositivo y la información de detección de movimiento.

Almacenamiento SDK

Almacenamiento SDK ofrece la posibilidad de integrar sistema de almacenamiento posibles. Permite a los desarrolladores leer de o escribir en cualquier lugar de almacenamiento de información: nube local, remota e incluso uno. La creación de un plugin de almacenamiento requiere implementar funciones estándar tales como: stream de I/O, si existe archivo, Borrar archivo, lista de archivos en la carpeta, etcetera. Almacenamiento SDK contiene un ejemplo de uso de un servidor FTP como una ubicación de almacenamiento.
