Generador de eventos genéricos

Un Evento Genérico es un método de envío de eventos de sistemas de terceros a un Sistema, que se puede utilizar para activar acciones del sistema en el motor de eventos y reglas. El Generador de Evento Genérico es una herramienta que ayuda a construir llamadas HTTP de Eventos Genérico.

Servidor API

Servidor API proporciona un conjunto estándar de llamadas API para sistemas de terceros. Da a los desarrolladores la capacidad para acceder a casi cualquier función de sistema disponible en el cliente regular: consulta y administrar los recursos del sistema (servidores, cámaras, usuarios, permisos), tirar video vivo o grabado fuera del sistema, generar eventos, crear normas, controlar cámaras PTZ y mucho más.

Panel de detalles de la cámara

Muestra información adicional al desarrollador para cámaras en la interfaz Web Admin:

  • ID de cámara
  • Enlaces para todos flujos de vídeo disponibles
  • Exportar vídeo directamente desde el navegador

Para abrir el Panel de Detalles de la Cámara: navegue hasta la interfaz Web Admin, elija Ver , seleccione la cámara de destino, haga clic en el ícono de configuración (gear), luego elija "Detalle de Cámaras"

SDK de Integración de Fuente de Video

SDK de Integración de Fuente de Video ofrece la posibilidad de integrar sistema de prácticamente cualquier fuente de vídeo grabado o en directo (cámaras IP, NVR, DVR, etcetera). El SDK permite crear métodos para descubrir, visualizar, analizar y grabar video. También permite la integración de puertos de E/S de dispositivos y la información de detección de movimiento.

SDK de Almacenamiento

SDK de Almacenamiento ofrece la posibilidad de integrar sistema de almacenamiento posibles. Permite a los desarrolladores leer de o escribir en cualquier lugar de almacenamiento de información: nube local, remota e incluso uno. La creación de un plugin de almacenamiento requiere implementar funciones estándar tales como: stream de I/O, si existe archivo, Borrar archivo, lista de archivos en la carpeta, etcetera. Almacenamiento SDK contiene un ejemplo de uso de un servidor FTP como una ubicación de almacenamiento.
