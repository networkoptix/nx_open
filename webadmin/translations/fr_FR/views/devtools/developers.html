Générateur d’événements génériques

Un événement générique est une méthode d'envoie d'événements depuis des systèmes tiers vers un système, qui peut être utilisé pour déclencher des actions système dans le moteur d'événements et de règles. Le Générateur d'événements génériques est un outil qui aide les développeurs et les intégrateurs à construire et à tester les appels d'Événement Générique HTTP.

API Serveur

Une API REST qui donne aux développeurs la possibilité d’accéder à presque toutes les fonctionnalités système disponibles dans le client normal: interroger / gérer les ressources système (serveurs, caméras, utilisateurs, autorisations), extraire une vidéo en direct ou enregistrée du système, générer des événements, créer règles, contrôlez les caméras PTZ, etc.

Panneau de détails caméra

Affiche des informations supplémentaires utiles aux développeurs pour des caméras individuelles dans l'onglet Afficher de l'interface Web Admin:

  • ID Caméra
  • Liens pour tous les flux vidéo disponibles
  • Exportation directe de vidéos depuis le navigateur

Pour ouvrir le panneau Détails de la caméra - accédez à l'interface Web Admin, choisissez Voir , sélectionnez la caméra cible, cliquez sur l'icône des paramètres (engrenage), puis choisissez "Détails caméra"

Source vidéo SDK

Le SDK source vidéo offre la possibilité d'intégrer presque toutes les sources vidéo en direct ou enregistrée (caméras IP, NVR, enregistreurs vidéo numériques, etc.) dans le Système et permet de créer des méthodes de découverte, l'affichage, l'analyse et l' enregistrement vidéo. Il permet également d'intégrer les ports d'E/S et périphériques, ainsi que d'extraire les informations de détection de mouvement de périphériques tiers.

Stockage SDK

Le SDK de stockage permet aux développeurs de lire ou d’écrire sur n’importe quel emplacement de stockage: local, distant et même nuage. La création d'un plug-in de stockage nécessite la mise en œuvre de fonctions standard telles que: flux d'E/S, si le fichier existe, suppression du fichier, liste des fichiers du dossier, etc. Le SDK de stockage contient un exemple d'utilisation d'un serveur FTP comme emplacement de stockage.
