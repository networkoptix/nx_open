Générateur d’événements génériques

Événements génériques est un type spécial d'événements pour les systèmes tiers afin de déclencher des actions dans le VMS. Pour créer un système intégré d'événement générique, il est nécessaire de faire une demande HTTP au serveur (en utilisant l'API du serveur). Actions for Generic Events can be configured using Events Rules dialog in desktop client application.

API Serveur

Serveur API fournis un standrad réglé pour appel API de systèmes de 3e partie. Il donne aux développeurs la possibilité d’accéder à presque chaque fonctionnalité système disponible dans le client régulier : requête/gérer les ressources système (serveurs, caméras, utilisateurs, permissions), accès au direct ou vidéo enregistrée hors système, génèrer des événements, créer des règles, contrôler les caméras PTZ et bien plus encore.

Panneau de détails caméra

Panneau spécial pour un webclient qui affiche de l'informations supplémentaires pour les caméras:

  • ID Caméra
  • Liens pour tous les flux vidéo disponibles
  • Exportation directe de vidéos depuis le navigateur

Pour ouvrir le panneau Détails de caméra - ouvrir client Web , Sélectionnez caméra, icône de réglage (vitesse), puis choisissez "Détails de la caméra"

Intégration de vidéo source SDK

Intégration de vidéo source SDK fournit l'abilité à intégrer au système pratiquement n’importe quelle source vidéo direct ou en différé (caméras IP, NVR, DVR, etc.). SDK permet de créer des méthodes pour découvrir, afficher, analyser et enregistrer vidéo. Il permet également d'intégrer des ports I/O de périphérique et les informations de détection de mouvement.

Stockage SDK

Stockage SDK fournit l'abilité potentiel d'intégrer l'info stockage système. Il permet aux développeurs de lire ou d’écrire sur n'importe quel emplacement de stockage: local, distant ou même nuage. Créer un plugin de stockage nécessite la mise en œuvre de fonctions standard telles que : flux e/s, si le fichier existe, supprimer le fichier, liste des fichiers dans le dossier, etc.. Stockage SDK contient également un exemple d’utilisation d’un serveur FTP comme un emplacement de stockage.
