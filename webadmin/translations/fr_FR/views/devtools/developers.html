Générateur d’événements génériques

Un événement générique est une méthode d'envoi d'événements de systèmes tiers vers un système qui peut être utilisé pour déclencher des actions système dans le moteur Événements et Règles. Le Générateur d'événements génériques est un outil qui aide les développeurs et les intégrateurs à construire et à tester les appels d'Événement Générique HTTP.

API Serveur

Une API REST qui donne aux développeurs la possibilité d'accéder à presque toutes les fonctionnalités système disponibles dans le client régulier : requête / gestion des ressources système (serveurs, caméras, utilisateurs, autorisations), tirez la vidéo en direct ou enregistrée hors du système, générer des événements, créer des règles, contrôler des caméras PTZ, et plus encore.

Panneau de détails caméra

Affiche des informations supplémentaires utiles aux développeurs pour des caméras individuelles dans l'onglet Afficher de l'interface Web Admin:

  • ID de la caméra
  • Liens pour tous les flux vidéo disponibles
  • Exportation directe de vidéos depuis le navigateur

Pour ouvrir le panneau Détails de la caméra - accédez à l'interface Web Admin, choisissez Voir , sélectionnez la caméra cible, cliquez sur l'icône des paramètres (engrenage), puis choisissez "Détails caméra"

Source vidéo SDK

Le SDK source vidéo offre la possibilité d'intégrer presque toutes les sources vidéo en direct ou enregistrée (caméras IP, NVR, enregistreurs vidéo numériques, etc.) dans le Système et permet de créer des méthodes de découverte, l'affichage, l'analyse et l' enregistrement vidéo. Il permet également d'intégrer les ports d'E/S et périphériques, ainsi que d'extraire les informations de détection de mouvement de périphériques tiers.

Stockage SDK

Le SDK de Stockage permet aux développeurs de lire ou d'écrire sur n'importe quel emplacement de stockage : local, distant, et même sur le cloud. La création d'un plugin de stockage nécessite l'implémentation de fonctions standard telles que: flux I/O, si le fichier existe, supprimez le fichier, la liste des fichiers dans le dossier, etc. Le SDK de stockage contient un exemple pour utiliser un serveur FTP comme emplacement de stockage.
