Documentation serveur

Ces paramètres sont accessibles dans:

Pour Windows dans la clé de registre: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Pour toutes les autres plates-formes, il est placé dans un fichier de configuration : /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. Pour commenter temporairement un paramètre , ajoutez le symbole de soulignement ("_") au début de son nom, par exemple "absoluteRtcpTimestamps=false" se convertit en "_absoluteRtcpTimestamps=false".
