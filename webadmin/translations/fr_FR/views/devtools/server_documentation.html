Documentation du serveur

Ces paramètres sont accessibles à:

Pour Windows dans la clé de registre : HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Serveur multimédia

Pour toutes les autres plates-formes, il est placé dans un fichier de configuration : /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. Pour commenter temporairement certains paramètres , ajoutez le symbole de tiret bas ("_") au début de son nom, par exemple "absoluteRtcpTimestamps=false" convertit en "_absoluteRtcpTimestamps=false".
