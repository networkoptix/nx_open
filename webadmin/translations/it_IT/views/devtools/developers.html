Generatore di eventi generici

Un evento generico è un metodo di invio di eventi da sistemi di terze parti a un sistema Wave, che può essere utilizzato per attivare azioni di sistema nel motore Eventi e regole. Generic Event Generator è uno strumento che aiuta gli sviluppatori e gli integratori a creare e a verificare le chiamate di eventi generici HTTP.

Server API

Offre agli sviluppatori la possibilità di accedere a quasi tutte le funzionalità di sistema disponibili nel client normale: ricerca / gestione delle risorse di sistema (server, telecamere, utenti, autorizzazioni), estrazione di video in diretta o registrati dal sistema, generazione di eventi, creazione di regole, controllo delle telecamere PTZ e altro ancora.

Pannello Dettagli telecamera

Visualizza informazioni aggiuntive per gli sviluppatori per le telecamere nell'interfaccia Web Admin:

  • ID della telecamera
  • Collegamenti per tutti i flussi video disponibili
  • Esportazione diretta di video dal browser

Per aprire il pannello Dettagli telecamera: accedere all'interfaccia Web Admin, selezionare Visualizza , seleziona la telecamera, fai clic sull'icona delle impostazioni (ingranaggio), quindi seleziona "Dettagli telecamera"

SDK sorgente video

La sorgente video SDK offre la possibilità di integrare quasi tutte le sorgenti video live o registrate (telecamere IP, NVR, DVR, ecc.) nel sistema e consente di creare metodi per ricercare, visualizzare, analizzare e registrare video. Permette anche l'integrazione di porte e dispositivi I / O, oltre a trarre informazioni di rilevamento del movimento da dispositivi di terze parti.

SDK di archiviazione

L'SDK di archiviazione consente agli sviluppatori di leggere o scrivere in qualsiasi posizione di archiviazione: locale, remoto e persino sul cloud. La creazione di un plug-in di archiviazione richiede l'implementazione di funzioni standard quali: flusso I / O, se il file esiste, elimina il file, visualizzare l'elenco dei file nella cartella, ecc. L'SDK di archiviazione contiene un esempio per l'utilizzo di un server FTP come percorso di archiviazione.
