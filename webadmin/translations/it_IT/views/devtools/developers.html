Generatore di eventi generici

Un evento generico è un metodo di invio di eventi da sistemi di terze parti a un sistema Wave, che può essere utilizzato per attivare azioni di sistema nel motore Eventi e regole. Generic Event Generator è uno strumento che consente di creare chiamate di eventi generici HTTP.

Server API

Server API fornisce un set standard di chiamate API per sistemi di terze parti. Offre agli sviluppatori la possibilità di accedere a quasi tutte le funzionalità di sistema disponibili nel client normale: query / gestione delle risorse di sistema (server, telecamere, utenti, autorizzazioni), estrazione di video in diretta o registrati dal sistema, generazione di eventi, creazione di regole, controllo delle telecamere PTZ e altro ancora.

Pannello Dettagli telecamera

Visualizza informazioni aggiuntive per gli sviluppatori per le telecamere nell'interfaccia Web Admin:

  • ID della telecamera
  • Collegamenti per tutti i flussi video disponibili
  • Esportazione diretta di video dal browser

Per aprire il pannello Dettagli telecamera: accedere all'interfaccia Web Admin, selezionare Visualizza , seleziona la telecamera, fai clic sull'icona delle impostazioni (ingranaggio), quindi seleziona "Dettagli telecamera"

SDK di integrazione della sorgente video

Video Source Integration SDK offre la possibilità di integrare praticamente qualsiasi sorgente video live o registrata (telecamere IP, NVR, DVR, ecc.) nel sistema. SDK rende possibile creare metodi per scoprire, visualizzare, analizzare e registrare video. Consente inoltre di integrare le porte I / O del dispositivo e le informazioni di rilevamento del movimento.

SDK di archiviazione

Storage SDK offre la possibilità di integrare potenziali storage nel sistema. Consente agli sviluppatori di leggere o scrivere in qualsiasi posizione di archiviazione: locale, remoto e persino cloud-one. La creazione di un plug-in di archiviazione richiede l'implementazione di funzioni standard quali: flusso I / O, se il file esiste, elimina file, elenco di file nella cartella, ecc. SDK di archiviazione contiene anche un esempio per l'utilizzo di un server FTP come percorso di archiviazione.
