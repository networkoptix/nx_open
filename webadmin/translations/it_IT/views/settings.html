Si prega di collegarlo alla rete locale o Internet Questo dispositivo ha una configurazione di rete errata.
Collegalo alla rete o impostare un indirizzo IP corretto.

Porta

Reimposta server

Questa azione ripristinerà questo server ai valori predefiniti (cancella il database e ripristina la configurazione). Il server verrà separato dal sistema corrente. L'archivio video rimarrà intatto.

Scollegare il server dal sistema

Questa azione separerà il server dal sistema mantenendo il database, gli utenti locali e la configurazione. L'archivio video rimarrà intatto. Gli utenti cloud saranno cancellati.

Riavvia

Cambia password amministratore

Se System è locale (non connesso al cloud) questo cambierà la password per l'utente "admin" e la password di ripristino (root) su alcuni server.
In caso contrario, cambierà solo la password di ripristino.
