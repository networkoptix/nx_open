Questo dispositivo è progettato per funzionare con l'unità disco fisso installata.
Spegni il dispositivo e installa un'unità disco fisso. Questo dispositivo necessita di una connessione di rete per funzionare.
Si prega di collegarlo alla rete locale o Internet Questo dispositivo ha una configurazione di rete errata.
Collegalo alla rete o impostare un indirizzo IP corretto.

Porta

Reimposta server

Questa azione ripristinerà questo server ai valori predefiniti (cancella il database e ripristina la configurazione). The server will be detached from the current System. L'archivio video rimarrà intatto.

Detach server from the System

This action will detach server from the System keeping database, local users and configuration. L'archivio video rimarrà intatto. Gli utenti cloud saranno cancellati.

Riavvia

Cambia password amministratore

Se System è locale (non connesso al cloud) questo cambierà la password per l'utente "admin" e la password di ripristino (root) su alcuni server.
In caso contrario, cambierà solo la password di ripristino.
