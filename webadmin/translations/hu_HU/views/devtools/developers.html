Általános esemény generátor

A Generic Event is a method of sending events from 3rd party Systems to a System, which can be used to trigger System actions in the Events & Rules engine. Az általános eseménygenerátor egy eszköz ami HTTP általános eseményhívások létrehozását segíti.

Szerver API

Server API provides a standard set of API calls for 3rd party Systems. It gives developers the ability to access almost every System feature available in the regular client: query/manage System resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

Kamera részletek panel

További fejlesztőbarát információkat jelenít meg a kamerákhoz a Web admin felületen:

  • Kamera ID
  • Link az össze elérhető videó streamhez
  • Közvetlen videó exportálás a böngészőből

A kamera részletek panel megnyitásához - navigáljon a Web admin felülethez, válassza a Nézet pontot, válassza ki a kívánt kamerát, kattintson a beállításikonra (fogaskerék), majd válassza a "Kamera részletek" pontot.

Videóforrás integráció SDK

Videóforrás integráció SDK lehetőséget biztosít minden élő, vagy rögzített videóforrás (IP kamera, NVR, DVR, stb...) integrációjára a rendszerbe. SDK lehetővé teszi videó rögzítő, felderítő, megjelenítő és elemző metódusok létrehozását. Szintén lehetőséget biztosít eszközök I/O portjainak és mozgásérzékelési információk integrálására.

Tároló SDK

A tároló SDK külső tárolók integrálását teszi lehetővé. Biztosítja a fejlesztők számára bármilyen tároló használatát olvasásra vagy írásra, legyen az helyi, távoli vagy felhő. Tároló plugin készítéséhez implementálni kell az olyan alap funkciókat mint az I/O streamek, fájl létrehozás, törlés, mappalisták, stb. A tároló SDK lehetőséget biztosít FTP szerverek tárolóként való használatára is.
