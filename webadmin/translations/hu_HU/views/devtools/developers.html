Általános esemény generátor

Az általános esemény egy olyan módszer, amellyel eseményeket küldhet harmadik féltől származó rendszerektől a rendszerbe, s amely meghatározható a rendszerműveletek indítására az Események és szabályok menüben. A Általános Esemény Generátor egy olyan eszköz, amely segít a fejlesztőknek és az integrátoroknak a verzió felépítésében és a HTTP általános eseményhívások tesztelésében.

Szerver API

Lehetőséget biztosít a fejlesztőknekminden a hagyományos kliensben elérhető jellemző hozzáféréséhez: rendszer források (szerverek, kamerák, felhasználók, engedélyek), élő és rögzített videók lekérésére, események létrehozására, szabályok írására, PTZ kamerák vezérlésére, stb.

Kamera részletek panel

További fejlesztőbarát információkat jelenít meg a kamerákhoz a Web admin felületen:

  • Kamera azonosítója
  • Link az össze elérhető videó streamhez
  • Közvetlen videó exportálás a böngészőből

A kamera részletek panel megnyitásához - navigáljon a Web admin felülethez, válassza a Nézet pontot, válassza ki a kívánt kamerát, kattintson a beállításikonra (fogaskerék), majd válassza a "Kamera részletek" pontot.

Videó forráskód (SDK)

A Video Source SDK lehetővé teszi, hogy integráljon szinte bármilyen élő vagy rögzített videót (IP kamerák, NVR, DVR, stb.) a rendszerhez, és lehetővé tegye, hogy létrehozzon módszereket a felderítésre, bemutatásra, elemzésre és videofelvételre. Ezenkívül lehetővé teszi az I / O portok és eszközök integrálását, valamint a mozgás érzékelés információt a harmadik féltől.

Tároló SDK

A Storage SDK lehetővé teszi a fejlesztők számára, hogy bármely tárolóhelyről olvassanak vagy írhassanak: helyi, távoli és akár felhőből is. Tároló bővítmény létrehozásához olyan standard funkciók végrehajtása szükséges, mint például: I / O adatfolyam, ha létezik fájl, törölje a fájlt, a mappában lévő fájlok listáját stb. A Storage SDK példát tartalmaz egy FTP szerver tárhelyként történő használatára.
