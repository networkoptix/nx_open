Általános esemény generátor

Az általános esemény egy speciális esemény 3. rendszerek részére VMS műveletek indításához. Általános esemény létrehozásához az integrált rendszerben HTTP kérés szükséges a szerver felé (szerver API használatával). Az általános események műveletei az Események&Szabályok párbeszédablakban konfigurálhatók az asztali kliensalkalmazásban.

Szerver API

A szerver API, 3. rendszerekhez szükséges API hívás példákat tartalmaz. Lehetőséget biztosít a fejlesztőknek minden a hagyományos kliensben elérhető jellemző hozzáféréséhez: rendszer források (szerverek, kamerák, felhasználók, engedélyek), élő és rögzített videók lekérésére, események létrehozására, szabályok írására, PTZ kamerák vezérlésére, stb.

Kamera részletek panel

Speciális panel a webkliens számára, amely további kamera információk megjelnítésére szolgál:

  • Kamera ID
  • Link az össze elérhető videó streamhez
  • Közvetlen videó exportálás a böngészőből

A kamera részletek panel megnyitásához - nyissa meg a web klienst, váassza ki a kamerát, beállítások ikon (fogaskerék), majd válassza a "Kamera részletek" pontot

Videóforrás integráció SDK

Videóforrás integráció SDK lehetőséget biztosít minden élő, vagy rögzített videóforrás (IP kamera, NVR, DVR, stb...) integrációjára a rendszerbe. SDK lehetővé teszi videó rögzítő, felderítő, megjelenítő és elemző metódusok létrehozását. Szintén lehetőséget biztosít eszközök I/O portjainak és mozgásérzékelési információk integrálására.

Tároló SDK

A tároló SDK külső tárolók integrálását teszi lehetővé. Biztosítja a fejlesztők számára bármilyen tároló használatát olvasásra vagy írásra, legyen az helyi, távoli vagy felhő. Tároló plugin készítéséhez implementálni kell az olyan alap funkciókat mint az I/O streamek, fájl létrehozás, törlés, mappalisták, stb. A tároló SDK lehetőséget biztosít FTP szerverek tárolóként való használatára is.
