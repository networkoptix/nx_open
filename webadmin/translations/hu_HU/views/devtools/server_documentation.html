Server Documentation

These settings are accessible at:

For Windows in registry key: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\"{{Config.productName}}"\"{{Config.productName}} Media Server".
For instance: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Hanwha\Hanwha Media Server

For all other platforms it is placed in a config file: /opt/{{Config.productName}}/mediaserver/etc/mediaserver.conf
For instance: /opt/hanwha/mediaserver/etc/mediaserver.conf
