W systemie Windows w kluczu rejestru: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Dla wszystkich innych platform jest on umieszczony w pliku konfiguracyjnym: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. To temporarily comment out some setting, add the underscore symbol ("_") to the begining of its name, e.g. "absoluteRtcpTimestamps=false" converts to "_absoluteRtcpTimestamps=false".
