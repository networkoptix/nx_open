Generator zdarzeń

W systemie można wywoływać zdarzenia generowane, które dostarczane są przez inne systemy lub urządzenia, w celu przekazywania mu określonych informacji i/lub uruchamiania w nim automatycznych reakcji, zgodnie z konfiguracją zdarzeń i reguł. Generator zdarzeń jest narzędziem wspomagającym tworzenie poleceń HTTP w celu wywoływania określonych zdarzeń.

API serwera

API serwera zawiera standardowy zestaw wywołań API dla systemów innych firm. Umożliwia programistom dostęp do niemal każdej funkcji podstawowej aplikacji klienckiej: zarządzanie zasobami systemu (serwerami, kamerami, użytkownikami, uprawnieniami), uzyskiwanie wideo na żywo z kamer lub z archiwum nagrań, generowanie zdarzeń, tworzenie reguł, sterowanie kamerami PTZ i wiele innych.

Panel szczegółowy kamery

Wyświetla w interfejsie Web Admin dodatkowe informacje o kamerach przydatne dla programistów:

  • ID kamery
  • Linki dla wszystkich dostępnych strumieni wideo
  • Bezpośredni eksport wideo z przeglądarki

Aby otworzyć panel szczegółowy kamery - przejdź do interfejsu Web Admin, wybierz Widok, wybierz kamerę, kliknij ikonę ustawień (koło zębate), a następnie wybierz opcję "Szczegóły kamery"

Pakiet SDK do integrowania nowych źródeł wideo

Pakiet Video Source Integration SDK umożliwia zintegrowanie systemu z praktycznie każdym źródłem wideo generującym obraz na żywo lub do pliku (np. kamery IP, rejestratory itp.). SDK umożliwia tworzenie metod wykrywania, wyświetlania, analizowania i nagrywania wideo. Pozwala również na integrację detekcji ruchu oraz portów I/O wbudowanych w urządzeniach.

Pakiet SDK do integrowania pamięci masowych

Pakiet Storage SDK umożliwia zintegrowanie nietypowych pamięci masowych. Umożliwia programistom odczyt lub zapis nagrań w dowolnych magazynach: lokalnych i zdalnych, nawet w chmurze. Utworzenie wtyczki pamięci masowej wymaga zaimplementowania standardowych funkcji, takich jak strumieniowanie I/O, "jeśli plik istnieje", "usuń plik", "lista plików w folderze" itp. To SDK zawiera również przykład użycia serwera FTP jako magazynu danych.
