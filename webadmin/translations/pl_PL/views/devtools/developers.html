Generator zdarzeń

W systemie można wywoływać zdarzenia generowane, które dostarczane są przez inne systemy lub urządzenia, w celu przekazywania mu określonych informacji i/lub uruchamiania w nim automatycznych reakcji, zgodnie z konfiguracją zdarzeń i reguł. Generic Event Generator to narzędzie, które pomaga programistom i integratorom budować i testować ogólne wywołania zdarzeń HTTP.

Server API

REST API umożliwia programistom dostęp do niemal każdej funkcji podstawowej aplikacji klienckiej: zarządzanie zasobami systemu (serwerami, kamerami, użytkownikami, uprawnieniami), uzyskiwanie wideo na żywo z kamer lub z archiwum nagrań, generowanie zdarzeń, tworzenie reguł, sterowanie kamerami PTZ i wiele innych.

Panel szczegółowy kamery

Wyświetla w interfejsie Web Admin dodatkowe informacje o kamerach przydatne dla programistów:

  • Camera ID
  • Linki dla wszystkich dostępnych strumieni wideo
  • Bezpośredni eksport wideo z przeglądarki

Aby otworzyć panel szczegółowy kamery - przejdź do interfejsu Web Admin, wybierz Widok, wybierz kamerę, kliknij ikonę ustawień (koło zębate), a następnie wybierz opcję "Szczegóły kamery"

SDK źródła wideo

SDK źródła wideo zapewnia możliwość integracji niemal każdego źródła wideo na żywo lub nagranego (kamery IP, NVR, DVR itp.) z systemem i umożliwia tworzenie metod wykrywania, wyświetlania, analizowania i nagrywania wideo. Umożliwia także integrację portów i urządzeń I/O, a także informacji o wykrywaniu ruchu z urządzeń innych firm.

Pakiet SDK do integrowania pamięci masowych

Storage SDK pozwala programistom odczytywać lub zapisywać dane w dowolnej przestrzeni do archiwizacji danych: lokalnej, zdalnej, a nawet w chmurze. Tworzenie wtyczki do archiwizacji wymaga implementacji standardowych funkcji, takich jak: strumień I/O, jeśli plik istnieje, usunięcie pliku, lista plików w folderze itp. SDK archiwizacji zawiera przykład wykorzystania serwera FTP jako lokalizacji do zapisu.
