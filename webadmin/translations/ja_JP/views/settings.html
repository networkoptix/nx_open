サーバーをリセット

This action will revert this server to defaults (clear the database and reset configuration). The server will be detached from the current system. Video archive will remain intact.

Detach server from the system

This action will detach server from the system keeping database, local users and configuration. Video archive will remain intact. Cloud users will be deleted.

Restart

Change admin password

If System is local (not connected to the cloud) this will change the password for user "admin" and the recovery (root) password on some servers.
Otherwise this will change the recovery password only.
