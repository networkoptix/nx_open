一般的なイベント ジェネレーター

一般的なイベントは、VMS のアクションがトリガーするサードパーティ製システムのイベントの種類です。 一般的なイベントを作成するにはサーバー (Server API を使用) にHTTPリクエストを行います。 一般的イベントのアクションは、デスクトップクライアントアプリケーションの[イベントルール] を使用して設定できます。

サーバーAPI

サーバー APIは、サードパーティシステムの標準API呼び出しセットを提供します 正規クライアントで利用可能なほとんどすべてのシステムの機能にアクセスすることができます クエリ/管理システム リソース (サーバー、カメラ、ユーザー、権限)、ライブまたは録画したビデオのシステムからイベントを生成するルールを作成、PTZ カメラを制御します

カメラの詳細

カメラの追加情報を表示するWebクライアント用の特別なパネル：

  • カメラ ID
  • 利用可能なすべてのビデオストリームのリンク
  • ブラウザからビデオのエクスポート

カメラ詳細パネルを開くには - Webクライアント を開き、カメラを選択し、設定アイコン（歯車）を選択し、次に「カメラの詳細」を選択します。

ビデオ ソース インテグレーション SDK

ビデオ ソース インテグレーション SDK は、ライブまたは録画したビデオ ソース (IP カメラ、NVRs、Dvr など) をインテグレートする手段を提供します SDK は検索、表示、解析、ビデオ記録方法を作成することが可能です デバイス I/O ポートとモーション検出情報をインテグレートすることができます

ストレージ SDK

ストレージ SDK は、システムに潜在的なストレージをインテグレートする機能を提供します それにより、開発者は、読み取りまたは任意のストレージの場所に書込み: ローカル、リモート、クラウド ストレージプラグインを作成するには、I / Oストリーム、ファイルが存在する場合はファイルの削除、フォルダ内のファイルのリストなどの標準機能を実装する必要があります　Storage SDKには、FTPサーバーを格納場所として使用する例も含まれています
