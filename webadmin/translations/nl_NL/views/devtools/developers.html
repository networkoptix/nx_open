Generic Event Generator

Een Generic Event is een methode om gebeurtenissen van systemen van derden naar een systeem te verzenden en die kan worden gebruikt om systeemdoelen, die aangemaakt zijn onder de camera regels, te activeren. De Generic Event Generator is een hulpmiddel waarmee u HTTP Generic Event API calls kunt aanmaken, welke u eenvoudig kunt knippen en plakken in het apparaat dat het Generic Event uit dient te sturen.

Server API

De server API bied een standaard set van API calls welke ingevoerd kunnen worden in systemen van derden (LPR, toegangscontrole, barcodescanners, Video Content Analyse, etc). Het geeft ontwikkelaars de mogelijkheid om bijna elk systeem-functie die beschikbaar is in de client: systeembronnen (servers, camera's, gebruikers, machtigingen) queries, beheren, live of opgenomen video genereren uit het systeem, events, regels maken, controle van de PTZ-camera's, en meer.

Camera Detail Paneel

Geeft voor de ontwikkelaar extra informatie weer over de camera's in de Web Admin-interface:

  • Camera ID
  • Links met alle beschikbare video-streams
  • Directe export van video via de webclient

Om het Camera Details Panel te openen - ga naar de Web Admin interface, selecteer Weergave , selecteer de doelcamera, klik op het instellingenpictogram (tandwieltje) en kies vervolgens "Camera details"

Video Source Integratie SDK

Video Source integratie SDK biedt de mogelijkheid om vrijwel elke live of opgenomen videobron (IP-camera's, NVRs, DVRs, enz.) te integreren in het systeem. SDK maakt het mogelijk om methoden voor het ontdekken van, weergeven, analyseren en opnemen van video. Ook is het hiermee mogelijk om, bijvoorbeeld, I/O modules en bewegingsdetectie te integreren.

Storage SDK

De storage SDK biedt de mogelijkheid om diverse opslagmogelijkheden te integreren in het systeem. Het staat ontwikkelaars te lezen van of schrijven naar iedere opslaglocatie: lokaal, extern of in de cloud. Het maken van een opslag plugin vereist de implementatie van standaardfuncties zoals: I/O stream, als bestand bestaat, verwijder bestand, lijst met bestanden in de map, enz. Opslag SDK bevat ook een voorbeeld voor het gebruik van een FTP-server als opslaglocatie.
