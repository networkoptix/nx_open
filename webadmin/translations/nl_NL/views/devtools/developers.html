Generic Event Generator

Een Generic Event is een methode om gebeurtenissen van systemen van derden naar een systeem te verzenden en die kan worden gebruikt om systeemdoelen, die aangemaakt zijn in de events en regels engine, te activeren. De Generic Event Generator is een tool die ontwikkelaars en integrators helpt bij het creeën van en het testen van HTTP Generic Events.

Server API

Een REST API die ontwikkelaars de mogelijkheid geeft om toegang te krijgen tot bijna elke systeemfunctie die beschikbaar is in de reguliere client: systeembronnen opvragen / beheren (servers, camera's, gebruikers, machtigingen), live of opgenomen video uit het systeem halen, evenementen genereren, creëren van regels, PTZ-camera's bedienen en meer.

API Testing Tool (nieuw) - interactieve documentatie en tools voor het testen van API calls
API Wijzigingenlog - een overzicht van de belangrijkste wijzigingen in de API sinds de laaste release
API Documentatie - een overzicht van de beschikbare Server API calls

Camera Details Paneel

Geeft voor de ontwikkelaar extra informatie weer over de camera's in de Web Admin-interface:

  • Camera ID
  • Links met alle beschikbare video-streams
  • Directe export van video via de webclient

Om het Camera Details Panel te openen - ga naar de Web Admin interface, selecteer Weergave , selecteer de doelcamera, klik op het instellingenpictogram (tandwieltje) en kies vervolgens "Camera details"

Video Source SDK

De Video Source SDK biedt de mogelijkheid voor het integreren van bijna elke live of opgenomen video-bron (IP-camera's, NVR's, DVR's, etc) in het systeem en maakt het mogelijk om methoden te creëren voor het ontdekken, het weergeven, analyseren en het opnemen van video. Het maakt ook de integratie van I/O-poorten en andere apparaten mogelijk, evenals het ophalen van bewegingsdetectie-informatie van externe apparaten.

Storage SDK

Met de Storage SDK kunnen ontwikkelaars lezen van of schrijven naar elke opslaglocatie: lokaal, op afstand of in de cloud. Het maken van een opslagplugin vereist implementatie van standaard functies zoals: I/O stream, if files exist, delete file, list of files in the folder, etc. De Storage SDK bevat een voorbeeld voor het gebruik van een FTP server als opslaglocatie.
