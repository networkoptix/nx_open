Sunucu Dökümantasyonları

Bu ayarlara şu adresten erişilebilir:

Windows: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Diğer tüm platformlar için bir yapılandırma dosyasına yerleştirilir: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. To temporarily comment out some setting, add the underscore symbol ("_") to the begining of its name, e.g. "absoluteRtcpTimestamps=false" converts to "_absoluteRtcpTimestamps=false".
