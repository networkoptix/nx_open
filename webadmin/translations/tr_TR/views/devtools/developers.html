Generic Events Generator

Genel Bir Olay, üçüncü parti sistemlerden Olaylar ve Kurallar motorundaki Sistem eylemlerini tetiklemek için kullanılabilecek bir Sistem'e olay gönderen bir yöntemdir. Genel Etkinlik Oluşturucu, geliştiricilerin ve tümleştiricilerin oluşturmasına ve HTTP Genel Etkinlik çağrılarını test etmesine yardımcı olan bir araçtır.

Sunucu API

It gives developers the ability to access almost every system feature available in the regular client: query/manage system resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

Kamera Ayrıntıları Paneli

Web Yöneticisi arayüzünde kameralar için geliştirici dostu ek bilgileri görüntüler:

  • Kamera ID
  • Kullanılabilir tüm video akışları için bağlantılar
  • Direct export video from browser

To open the Camera Details Panel - navigate to the Web Admin interface, choose View, select the target camera, click on the settings icon (gear), then choose "Camera Details"

Video Source SDK

Video Kaynağı SDK, hemen hemen tüm canlı veya kaydedilmiş video kaynaklarını (IP Kameralar, NVR'ler, DVR'ler, vb.) Sisteme entegre etme yeteneği sağlar ve keşfedilmesi, görüntülenmesi, analiz edilmesi ve video kaydetmek için yöntemler oluşturmayı mümkün kılar. Ayrıca, I / O portlarını ve cihazlarını entegre etmenin yanı sıra, 3. parti cihazlardan hareket algılama bilgisini de çekmeyi sağlar.

Depolama için SDK

Storage SDK, geliştiricilerin herhangi bir depolama konumundan okumalarını veya yazmalarını sağlar: yerel, uzak ve hatta buluttan. Depolama eklentisi oluşturmak için bazı fonksiyonları kullanmak gerekebilir: I/O stream, if file exist, delete file, list of files in the folder, etc. Storage SDK FTP kullanımı ile ilgili bazı örnekler içerir Sunucu depo olarak kullanılabilir.
