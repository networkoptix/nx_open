Serverdokumentation

Diese Einstellungen sind abrufbar unter:

Für Windows im Registrierungsschlüssel: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Für alle anderen Plattformen wird es in einer Konfigurationsdatei gespeichert: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. Um eine Einstellung vorübergehend zu kommentieren, fügen Sie am Anfang des Namens das Unterstrichsymbol ("_") hinzu, z. B. "absoluteRtcpTimestamps = false" wird zu "_absoluteRtcpTimestamps = false" konvertiert.
