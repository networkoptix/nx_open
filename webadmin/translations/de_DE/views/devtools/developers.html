Generic Events Generator

Generic Events is a special kind of events for 3rd party systems to trigger actions in the VMS. To create Generic Event integrated system needs to make HTTP request to the Server (using Server API). Actions for Generic Events can be configured them using Events&Rules dialog in desktop client application.

Server API

Server API provides a standard set of API calls for 3rd party systems. It gives developers the ability to access almost every system feature available in the regular client: query/manage system resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

Camera Details Panel

Special panel for a webclient which displays additional information for cameras:

  • Camera ID
  • Links for all available video-streams
  • Direct export video from browser

To open Camera Details Panel - open web client, select camera, settings icon (gear), then choose "Camera Details"

Video Source Integration SDK

Die Video Source Integration SDK bietet die Möglichkeit praktisch jede Live- oder aufgezeichnete Videoquelle (IP-Kameras, NVRs, DVRs, etc.) ins System zu integrieren. SDK ermöglicht es Methoden zur Entdeckung, Anzeige, Analyse und Videoufzeichnung zu erstellen. Es erlaubt auch die Integration von Geräte-I/O-Ports und Bewegungserkennungsinformationen.

Storage SDK

Storage SDK bietet die Möglichkeit potentielle Speicher ins System zu integrieren. Es ermöglicht Entwicklern das Lesen oder Schreiben auf jeden Speicherort: Lokal, Remote und sogar über die Cloud. Das Erstellen eines Speicherplugins erfordert die Implementierung von Standardfunktionen wie: I/O-Stream, falls Datei vorhanden ist, Datei löschen, Liste der Dateien im Ordner usw. Storage SDK enthält auch ein Beispiel für die Verwendung eines FTP-Servers als Speicherort.
