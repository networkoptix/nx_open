Disse innstillingene er tilgjengelige på:

For Windows i registernøkkel: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

For alle andre plattformer plasseres den i en config-fil: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. To temporarily comment out some setting, add the underscore symbol ("_") to the begining of its name, e.g. "absoluteRtcpTimestamps=false" converts to "_absoluteRtcpTimestamps=false".
