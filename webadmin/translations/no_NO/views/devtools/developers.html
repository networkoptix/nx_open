Generisk hendelse generator

En gnerisk hendelse er en metode for å sende hendelser fra tredjepartssystemer til et system, som kan brukes til å utløse systemhandlinger i hendelser og regler-motoren. Generisk hendelse generator er et verktøy som bidrar til å bygge HTTP Generic Event kommandoer.

Server API

Server API gir et standard sett med API-anrop for tredjepartssystemer. Det gir utviklere muligheten til å få tilgang til nesten alle systemfunksjoner som er tilgjengelige i den vanlige klienten: spørre / administrere systemressurser (servere, kameraer, brukere, tillatelser), trekk live eller innspilt video ut av System, generer hendelser, opprett regler, kontroll PTZ kameraer og mer.

Kameradetaljer Panel

Viser tilleggsutvikler-vennlig informasjon for kameraer i Web Admin-grensesnittet:

  • Kamera-ID
  • Lenker for alle tilgjengelige videostrømmer
  • Direkte videoeksport fra nettleser

For å åpne Kameradetaljer-panelet - naviger til Web Admin-grensesnittet, velg Vis , velg målkamera, klikk på innstillingsikonet (gir), og velg deretter "Kameradetaljer"

Video Source Integration SDK

Video Source Integration SDK gir muligheten til å integrere nesten alle levende eller innspilte videokilder (IP-kameraer, NVRer, DVRer, etc.) i System. SDK gjør det mulig å lage metoder for å oppdage, vise, analysere og ta opp video. Det tillater også integrering av enhetens I / O-porter og bevegelsesdeteksjon informasjon.

Lagrings SDK

Storage SDK gir muligheten til å integrere potensiell lagring i System. Det lar utviklere lese fra eller skrive til hvilken som helst lagringsplass: lokal, ekstern og til og med sky en. Å lage et plugin for lagring krever at du implementerer standardfunksjoner som: I / O-strøm, hvis filen eksisterer, slett fil, liste over filer i mappen, etc.Storage SDK inneholder også et eksempel på bruk av en FTP-server som lagringsplass.
