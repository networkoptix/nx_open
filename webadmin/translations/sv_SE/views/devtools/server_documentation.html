Serverdokumentation

Dessa inställningar är tillgängliga på:

För Windows i registernyckel: HKEY_LOCAL_MACHINE \ SOFTWARE \{VMS Vendor}\{VMS Vendor} Media Server

För alla andra plattformar placeras den i en config-fil: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. To temporarily comment out some setting, add the underscore symbol ("_") to the begining of its name, e.g. "absoluteRtcpTimestamps=false" converts to "_absoluteRtcpTimestamps=false".
