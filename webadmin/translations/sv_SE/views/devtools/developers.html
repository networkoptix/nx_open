Generic Events Generator

En generisk händelse är en metod att skicka händelser från 3: e partsystem till ett System, som kan användas för att utlösa systemåtgärder i händelse & regelmotorn. Generic Event Generator är ett verktyg som hjälper utvecklare och integratörer att bygga och testa HTTP Generic Event calls.

Server API

Ett REST API som ger utvecklare möjlighet att få tillgång till nästan alla systemfunktioner som finns tillgängliga i den vanliga klienten: Fråga / hantera systemresurser (servrar, kameror, användare, behörigheter), dra live eller inspelad video från System, generera händelser, skapa regler, kontroll PTZ kameror och mer.

API-testverktyg (nytt) - Interaktiv dokumentation och verktyg för testning av API-samtal
API Changelog - En lista över bryta ändringar i API sedan den senaste utgåvan
API-dokumentationen - En fullständig lista över tillgängliga API-samtal från Server API

Panel för kameradetaljer

Visar ytterligare utvecklarvänlig information för kameror i Web Admin-gränssnittet:

  • Kamera-ID
  • Länkar för alla tillgängliga videoströmmar
  • Direktexportera video från webbläsare

För att öppna kameradetaljeringspanelen - navigera till webbadministratörsgränssnittet, välj Visa , välj målkamera, klicka på inställningsikonen (kugghjul) och välj sedan "Kamerainformation"

Videokälla SDK

Videokällans SDK ger möjlighet att integrera nästan alla levande eller inspelade videokällor (IP-kameror, NVR, DVR, etc.) i System och gör det möjligt att skapa metoder för att upptäcka, visa, analysera och spela in video. Det gör det också möjligt att integrera I / O-portar och enheter, samt att dra rörelsesdetektering information från tredje parts enheter.

Lagrings SDK

Lagrings SDK tillåter utvecklare att läsa från eller skriva till någon lagringsplats: lokal, fjärrkontroll och till och med moln en. Skapa ett lagringsplugin kräver implementering av standardfunktioner som: I / O-ström, om fil finns, radera fil, lista med filer i mappen etc. Storage SDK innehåller ett exempel för att använda en FTP server som en lagringsplats.
