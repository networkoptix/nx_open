Generic Events Generator

En generisk händelse är en metod att skicka händelser från 3: e partsystem till ett System, som kan användas för att utlösa systemåtgärder i händelse & regelmotorn. Den generiska Eventgeneratorn är ett verktyg som hjälper bygga HTTP Generic Event förfrågningar.

Server API

Server API tillhandahåller en standard uppsättning API-förfrågningar för tredje parts system. Det ger utvecklare möjlighet att få tillgång till nästan alla systemfunktioner som finns i den vanliga klienten: Fråga / hantera systemresurser (servrar, kamera, användare, behörigheter), hämta live eller inspelad video från System, skapa händelser, skapa regler, kontrollera PTZ-kameror och mer.

Panel för kameradetaljer

Visar ytterligare utvecklarvänlig information för kameror i Web Admin-gränssnittet:

  • Kamera-ID
  • Länkar för alla tillgängliga videoströmmar
  • Direktexportera video från webbläsare

För att öppna kameradetaljeringspanelen - navigera till webbadministratörsgränssnittet, välj Visa , välj målkamera, klicka på inställningsikonen (kugghjul) och välj sedan "Kamerainformation"

Integration av videokälla SDK

Videokälla Integration SDK ger möjlighet att integrera praktiskt taget alla live eller inspelade videokällor (IP-kameror, NVR, DVR, etc.) till System. SDK gör det möjligt att skapa metoder för att upptäcka, visa, analysera och spela in video. Det möjliggör också integrering av enhetens I/O-portar och information om rörelsedetektering.

Lagrings SDK

Lagring SDK ger möjlighet att integrera potentiell lagring i System. Det låter utvecklare läsa från eller skriva till någon lagringsplats: lokal, fjärr och till och med moln. Skapa ett plugin för lagringsutrymme kräver implementering av standardfunktioner som: I/O-ström, om filen finns, radera fil, lista med filer i mappen etc. LagringsSDK innehåller också ett exempel för att använda en FTP-server som en lagringsplats.
