Documentação do Servidor

Essas definições estão acessíveis em:

Para Windows na registry key: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Para todas as outras plataformas, é colocado em um ficheiro de configuração: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. To temporarily comment out some setting, add the underscore symbol ("_") to the begining of its name, e.g. "absoluteRtcpTimestamps=false" converts to "_absoluteRtcpTimestamps=false".
