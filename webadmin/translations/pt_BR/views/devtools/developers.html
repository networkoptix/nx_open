Gerador de Eventos Genéricos

Um Evento Genérico é um método de envio de eventos a partir sistemas de terceiros para um Sistema, que pode ser usado para accionar acções do Sistema no motor "Eventos & Regras". O gerador de eventos genéricos é uma ferramenta que ajuda a construir chamadas de eventos HTTP genéricos.

API do Servidor

A API do servidor fornece um conjunto padrão de chamadas de API para sistemas de terceiros. Ele dá aos desenvolvedores a capacidade de acessar quase todos os recursos do sistema disponíveis no cliente regular: consultar / gerenciar recursos do sistema (servidores, câmeras, usuários, permissões), extrair vídeos ao vivo ou gravados do Sistema, gerar eventos, criar regras, controlar câmeras PTZ e muito mais.

Painel de detalhes da câmera

Exibe informações adicionais favoráveis ​​ao desenvolvedor para câmeras na interface do Web Admin:

  • ID da câmera
  • Links para todos os fluxos de vídeo disponíveis
  • Vídeo de exportação direto do navegador

Para abrir o painel Detalhes da câmera - navegue até a interface Web Admin, escolha Visualizar, selecione a câmera alvo, clique no ícone de configurações (engrenagem) e escolha "Detalhes da câmera"

SDK de integração de fontes de vídeo

O SDK de integração de fonte de vídeo oferece a capacidade de integrar praticamente qualquer fonte de vídeo ao vivo ou gravada (Câmeras IP, NVRs, DVRs, etc) no sistema. O SDK possibilita a criação de métodos para descobrir, exibindo, analisando e gravando vídeos. Também permite integrar portas de E/S de dispositivos e detecção de movimento informação.

SDK de armazenamento

O Storage SDK fornece a capacidade de integrar o armazenamento potencial no sistema. Ele permite que aos desenvolvedores ler ou gravar em qualquer local de armazenamento: local, remoto e até mesmo em nuvem. Criando um armazenamento O plugin requer a implementação de funções padrão como: fluxo de E/S, se o arquivo existir, arquivo de exclusão, lista de arquivos na pasta, etc. O SDK Storage também contém um exemplo para usar um servidor FTP como um local de armazenamento.
