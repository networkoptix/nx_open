Gerador de Eventos Genéricos

Um Evento Genérico é um método de envio de eventos de sistemas de terceiros para um Sistema que pode ser usado para acionar ações do sistema no mecanismo Eventos e Regras. O Gerador de Eventos Genéricos é uma ferramenta que ajuda desenvolvedores e integradores a construir e testar chamadas de evento Genérico HTTP.

API do Servidor

A API REST que dá aos desenvolvedores a capacidade de acessar quase todos os recursos de sistema disponíveis no regulares cliente: query / gerenciar recursos do sistema (servidores, câmeras, usuários, permissões), puxe ao vivo ou gravado vídeo fora do sistema, gerar eventos, criar regras, controle de câmeras PTZ e muito mais.

Ferramenta de teste da API (nova) - documentação e ferramentas interativas para testar chamadas de API
Registro de alterações da API - uma lista de alterações recentes na API desde o último lançamento
Documentação da API - uma lista completa de chamadas disponíveis da API do servidor

Painel de detalhes da câmera

Exibe informações adicionais amigáveis para o desenvolvedor para câmeras individuais na guia de visualização da Interface de Administração da Web:

  • ID da câmera
  • Links para todos os fluxos de vídeo disponíveis
  • Vídeo de exportação direto do navegador

Para abrir o painel Detalhes da câmera - navegue até a interface Web Admin, escolha Visualizar, selecione a câmera alvo, clique no ícone de configurações (engrenagem) e escolha "Detalhes da câmera"

Fonte de vídeo SDK

O SDK de Fonte de Vídeo fornece a capacidade de integrar quase qualquer fonte de vídeo ao vivo ou gravada (Câmaras IP, NVRs, DVRs, etc) no sistema e permite criar métodos para descobrir, exibir, analisar e gravar vídeo. Permite também integrar portas e dispositivos I/O, bem como puxar detecção de movimento informações de dispositivos de terceiros.

SDK de armazenamento

O SDK de armazenamento permite que os desenvolvedores para ler ou escrever para qualquer local de armazenamento: local, remoto, e até mesmo uma nuvem. A criação de um plug-in de armazenamento requer a implementação de funções padrão, como: fluxo de E / S, se o arquivo existir, arquivo de exclusão, lista de arquivos na pasta, etc. O SDK de armazenamento contém um exemplo para usar um servidor FTP como local de armazenamento.
