Anna palautetta

Generic Events -generaattori

Generic Event on tapa lähettää tapahtumia kolmannen osapuolen järjestelmistä järjestelmään, jonka avulla voidaan käynnistää System-toimintoja Events & Rule-enginessä. Generic Event Generator on työkalu, joka auttaa luomaan HTTP-geneerisiä tapahtumia.

Generic Events -generaattori

Palvelin API

Palvelin-sovellusliittymä tarjoaa vakiomuotoisen API-kutsun kolmannen osapuolen järjestelmille. Se antaa kehittäjille pääsyn lähes kaikkiin järjestelmän normaalin client-sovelluksen ominaisuuksiin: tehdä kyselyitä / hallita järjestelmän resursseja (palvelimet, kamerat, käyttäjät, oikeudet), viedä videokuvaa tai tallennettua videota järjestelmästä, luoda tapahtumia, luoda sääntöjä, hallita PTZ-kameroita ja paljon muuta.

Kameran tiedot -paneeli

Näyttää ylimääräisiä Web-ylläpitäjän tietoja kameroista Web Admin-käyttöliittymässä:

  • Kameran tunnus
  • Linkit kaikkiin saatavilla oleviin videostreameihin
  • Suora videon vienti selaimen kautta

Avataksensi Kameran tiedot-paneelin - siirry Web-Admin käyttöliittymään, valitse Näytä , valitse kamera, paina asetus-kuvaketta (rattaan-kuva) ja valitse sitten "Kameran tiedot"

kameran tiedot -paneeli

Video Source Integration SDK

Video Source Integration SDK tarjoaa mahdollisuuden integroida käytännöllisesti katsoen kaikki live- tai tallennetut videolähteet (IP-kamerat, NVR: t, DVR: t jne.) Järjestelmään. SDK mahdollistaa keinoja löytää näyttää, analysoida tallennettua videota. Se mahdollistaa myös laitteen I / O-porttien ja liiketunnistuksen tietojen integroinnin.

hae SDK

Storage SDK

Storage SDK tarjoaa mahdollisuuden integroida tallennustilan järjestelmään. Sen avulla kehittäjät voivat lukea tai kirjoittaa mihinkä tahansa tallennuspaikkaan: paikallinen, kauko-, ja jopa pilvi yksi. Tallennuksen laajennuksen luominen vaatii standarditoimintojen, kuten: I / O-virran, jos tiedosto on olemassa, poistaminen, tiedostojen luettelo kansiosta, jne. Säilytys SDK sisältää myös esimerkin FTP-palvelimen käyttämisestä tallennuspaikkana.
