Anna palautetta

Generic Events -generaattori

Generic Event on tapa lähettää tapahtumia kolmannen osapuolen järjestelmistä järjestelmään, jonka avulla voidaan käynnistää System-toimintoja Events & Rule-enginessä. Generic Event Generator on työkalu, jonka avulla kehittäjät ja integraattorit voivat rakentaa ja testata HTTP-geneerisiä tapahtumia.

Palvelin API

REST-sovellusliittymä antaa kehittäjille mahdollisuuden käyttää lähes jokaista järjestelmän ominaisuutta Client-sovelluksella lähettää kyselyitä / hallita järjestelmän resursseja (palvelimet, kamerat, käyttäjät, oikeudet), hakea/reaaliaikaista tai tallennettua videota järjestelmästä, luoda tapahtumia, luoda sääntöjä, hallita PTZ-kameroita ja paljon muuta.

Kameran tiedot -paneeli

Näyttää lisää kehittäjäystävällisiä tietoja yksittäisille kameroille Web-ylläpitäjän käyttöliittymän Näytä-välilehdessä:

  • Kameran tunnus
  • Linkit kaikkiin saatavilla oleviin videostreameihin
  • Suora videon vienti selaimen kautta

Avataksensi Kameran tiedot-paneelin - siirry Web-Admin käyttöliittymään, valitse Näytä , valitse kamera, paina asetus-kuvaketta (rattaan-kuva) ja valitse sitten "Kameran tiedot"

kameran tiedot -paneeli

Videolähteen SDK

Video Source SDK tarjoaa mahdollisuuden integroida lähes minkä tahansa reaaliaikaisen tai tallennetun videolähteen (IP-kamerat, NVR, DVR, jne) järjestelmään ja tekee mahdolliseksi luoda menetelmiä löytää, näyttää, analysoida ja tallennettua kuvamateriaalia. Se mahdollistaa myös I/O-porttien ja -laitteiden integroinnin sekä liiketunnistuksen tietojen hakemisen kolmannen osapuolen laitteista.

Storage SDK

Tallennus SDK: n avulla kehittäjät voivat lukea tai kirjoittaa mihin tahansa tallennuspaikkaan: paikallisesti, etäkohteeseen ja jopa pilvi-palveluun. Tallennuslaajennuksen luominen edellyttää standarditoimintojen, kuten I/O-streamin, toteuttamista, jos tiedosto on jo olemassa, poista tiedosto, luettelo kansiossa olevista tiedostoista jne. Tallennus SDK sisältää esimerkin FTP -palvelimen käyttämisestä tallennuspaikkana.
