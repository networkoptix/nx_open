Generador d'esdeveniments genèrics

Un esdeveniment genèric és un mètode per enviar esdeveniments de sistemes de tercers a un sistema que es pot utilitzar per activar accions del sistema al motor d'esdeveniments i regles. El generador d'esdeveniments genèrics és una eina que ajuda els desenvolupadors i els integradors a crear i provar les crides d'esdeveniments genèrics HTTP.

Generador d'esdeveniments genèrics

API del servidor

Una API REST que ofereix als desenvolupadors la possibilitat d'accedir a gairebé totes les funcions del sistema disponibles al client normal: consulta/gestió dels recursos del sistema (servidors, càmeres, usuaris, permisos), extracció de vídeo en directe o enregistrat del sistema, generació d'esdeveniments, creació de regles, control de càmeres PTZ i molt més.

Video Source SDK

The Video Source SDK provides the ability to integrate almost any live or recorded video source (IP Cameras, NVRs, DVRs, etc) into System and makes it possible to create methods for discovering, displaying, analyzing and recording video. It also allows integrating I/O ports and devices, as well as pulling motion detection information from 3rd party devices.

get SDK

Storage SDK

The Storage SDK allows developers to read from or write to any storage location: local, remote, and even cloud one. Creating a storage plugin requires implementing standard functions such as: I/O stream, if file exist, delete file, list of files in the folder, etc. The Storage SDK contains an example for using an FTP server as a storage location.
