Generic Events Generator

Generic Events zijn een speciaal soort gebeurtenissen voor 3rd party systemen om acties in het VMS te activeren. Om generic events te creëren, dient het te integreren systeem een HTTP request te kunnen maken naar de server (met behulp van de Server API). Acties voor Generic Events kunnen worden geconfigureerd met behulp van Events & Rules engine in de desktop client applicatie.

Server API

De server API biedt een standaard set van API calls voor 3rd party systemen. Het geeft developers de mogelijkheid om toegang te verkrijgen tot elke systeem feature die beschikbaar is in de reguliere client: query/beheer systeem resources (servers, camera's, gebruikers, rechten), verkrijgen van stream van live en opgenomen beelden uit het systeem, het genereren van events, creëren van regels, PTZ camera's bedienen en nog veel meer.

Camera Details Paneel

Speciaal paneel voor een webclient die extra informatie voor camera's bevat:

  • Camera ID
  • Links met alle beschikbare video-streams
  • Directe export video vanaf de browser

Om het Camera Detail Paneel te openen - open web client, kies de camera, instellingen icoon (gear) en kies "Camera Details"

Video Source Integratie SDK

Video Source integratie SDK biedt de mogelijkheid om vrijwel elke live of opgenomen videobron (IP-camera's, NVRs, DVRs, enz.) te integreren in het systeem. SDK maakt het mogelijk om methoden voor het ontdekken van, weergeven, analyseren en opnemen van video. Het maakt het ook mogelijk om bijvoorbeeld I/O modules en Video Content Analyse te integreren.

download SDK

Storage SDK

De storage SDK biedt de mogelijkheid om diverse opslagmogelijkheden te integreren in het systeem. Het staat ontwikkelaars te lezen van of schrijven naar iedere opslaglocatie: lokaal, extern of in de cloud. Het maken van een opslag plugin vereist de implementatie van standaardfuncties zoals: I/O stream, als bestand bestaat, verwijder bestand, lijst met bestanden in de map, enz. Opslag SDK bevat ook een voorbeeld voor het gebruik van een FTP-server als opslaglocatie.
