Generieke gebeurtenissen Generator

A Generic Event is a method of sending events from 3rd party Systems to a System, which can be used to trigger System actions in the Events & Rules engine. De Generic Event Generator is een hulpmiddel waarmee u HTTP Generic Event API calls kunt aanmaken, welke u eenvoudig kunt knippen en plakken in het apparaat dat het Generic Event uit dient te sturen.

Server API

Server API provides a standard set of API calls for 3rd party Systems. It gives developers the ability to access almost every System feature available in the regular client: query/manage System resources (servers, cameras, users, permissions), pull live or recorded video out of System, generate events, create rules, control PTZ cameras, and more.

Camera Detail Paneel

Geeft voor de ontwikkelaar extra informatie weer over de camera's in de Web Admin-interface:

  • Camera ID
  • Links met alle beschikbare video-streams
  • Directe export van video via de webclient

Om het Camera Details Panel te openen - ga naar de Web Admin interface, selecteer Weergave , selecteer de doelcamera, klik op het instellingenpictogram (tandwieltje) en kies vervolgens "Camera details"

Video Source Integratie SDK

De Video Source integratie SDK biedt de mogelijkheid om vrijwel elke live of opgenomen videobron (IP-camera's, NVRs, DVRs, enz.) te integreren in het systeem. Deze SDK maakt het mogelijk om video te ontdekken, weer te geven, analyseren en op te nemen. Ook is het hiermee mogelijk om, bijvoorbeeld, I/O modules en bewegingsdetectie te integreren.

Storage SDK

De storage SDK biedt de mogelijkheid om diverse opslagmogelijkheden te integreren in het systeem. Het staat ontwikkelaars te lezen van of schrijven naar iedere opslaglocatie: lokaal, extern of in de cloud. Het maken van een opslag plugin vereist de implementatie van standaardfuncties zoals: I/O stream, als bestand bestaat, verwijder bestand, lijst met bestanden in de map, enz. Opslag SDK bevat ook een voorbeeld voor het gebruik van een FTP-server als opslaglocatie.
