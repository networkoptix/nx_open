Generieke gebeurtenissen Generator

Een Generic Event is een methode om gebeurtenissen van systemen van derden naar een systeem te verzenden en die kan worden gebruikt om systeemdoelen, die aangemaakt zijn onder de camera regels, te activeren. De Generic Event Generator is een hulpmiddel waarmee ontwikkelaars en integrators bouwen en HTTP Generic Event-oproepen kunnen testen.

Server API

Het geeft ontwikkelaars de mogelijkheid om toegang te verkrijgen tot elke systeemfunctie die beschikbaar is in de reguliere desktop client: query/beheer vsan systeembronnen (servers, camera's, gebruikers, rechten), verkrijgen van streams van de live en opgenomen beelden uit het systeem, het genereren van events, creëren van regels, PTZ camera's bedienen en nog veel meer.

Camera Detail Paneel

Bied voor de ontwikkelaar extra informatie weer over de camera's in de Web Admin-interface:

  • Camera ID
  • Links met alle beschikbare video-streams
  • Directe export van video via de webclient

Om het Camera Details Panel te openen - ga naar de Web Admin interface, selecteer Weergave , selecteer de doelcamera, klik op het instellingenpictogram (tandwieltje) en kies vervolgens "Camera details"

open Camera Detail Paneel

Video Bron SDK

De videobron SDK biedt de mogelijkheid bijna elke live of opgenomen videobron (IP-camera's, NVR's, DVR's, enz.) te integreren in System en maakt het mogelijk om methoden te creëren voor het ontdekken, weergeven, analyseren van en het opnemen van video. Het maakt ook integratie van I / O-poorten en apparaten mogelijk, evenals het ophalen van bewegingsdetectie informatie van apparaten van derden.

Storage SDK

Met de Storage SDK kunnen ontwikkelaars lezen van of schrijven naar elke opslaglocatie: lokaal, extern en zelfs cloud one. Het maken van een opslagplug-in vereist het implementeren van standaardfuncties zoals: I / O-stream, als bestand bestaat, bestand verwijderen, lijst met bestanden in de map, etc. De Storage SDK bevat een voorbeeld voor het gebruik van een FTP server als opslaglocatie.
