Server documentatie

Deze instellingen zijn toegankelijk via:

Voor Windows in registersleutel: HKEY_LOCAL_MACHINE\SOFTWARE\{VMS Vendor}\{VMS Vendor} Media Server

Voor alle andere platformen wordt het in een configuratiebestand geplaatst: /opt/{vmsvendor}/mediaserver/etc/mediaserver.conf. Om tijdelijk een instelling uit te commentaren, voegt u het onderstrepingsteken ("_") toe aan het begin van de naam, bijv. "absoluteRtcpTimestamps = false" wordt omgezet in "_absoluteRtcpTimestamps = false".
