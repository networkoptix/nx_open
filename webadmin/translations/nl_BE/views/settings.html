Dit apparaat is bedoeld om te werken met een harde schijf.
Gelieve het apparaat uit te schakelen en een harde schijf te installeren. Dit apparaat heeft een netwerkverbinding nodig om te functioneren.
Sluit het apparaat aanop het lokale netwerk of het internet Dit apparaat heeft de verkeerde netwerkconfiguratie.
Sluit het apparaat aan op het netwerk en stel het juist IP adres in.

Poort

Reset server

Met deze actie wordt deze server teruggezet naar de standaardinstellingen (de database en de configuratie worden gewist). The server will be detached from the current System. Videoarchief zal intact blijven.

Detach server from the System

This action will detach server from the System keeping database, local users and configuration. Videoarchief zal intact blijven. Alle cloud gebruikers zullen worden verwijderd.

Herstarten

Wijzig admin wachtwoord

Als het systeem niet is aangesloten op de cloud dan zal het wachtwoord voor de gebruiker "admin" and het herstel (root) wachtwoord gewijzigd worden op de servers.
Anders zal alleen het recovery wachtwoord gewijzigd worden.
