אירוע גנרי הוא שיטה לשליחת אירועים ממערכות צד שלישי למערכת, אשר ניתן להשתמש בהם כדי להפעיל רשימת אירועים וחוקים של מערכת ההקלטה. אשף אירוע גנרי הוא כלי אשר מסייע למפתחים ואינטגרטורים לבנות ולבדוק קריאות HTTP.

ממשק API של שרת

REST API אשר נותן למפתחים את האפשרות לגשת כמעט לכל פיצ'ר הקיים בתוכנת קליינט כגון: ניהול מערכת (התקנים, מצלמות, משתמשים והרשאות), משיכת תזרים וידוא או הקלטות מהמערכת, יצור ארועים או יצור חוקים, שליטה במצלמות PTZ ועוד.

חלון פרטי מצלמה

מציג מידע ידידותי נוסף על המצלמות למפתחים בממשק ה Web:

  • מזהה מצלמה
  • קישורים לכל זרמי הווידאו הזמינים
  • ייצוא וידאו ישירות מתוך הדפדפן

כדי לפתוח את החלונית פרטי המצלמה - נווט אל ממשק מנהל הרשת, בחר תצוגה , בחר את המצלמה היעד, לחץ על סמל ההגדרות (גלגל שיניים), ולאחר מכן בחר "פרטי מצלמה"

פתח את לוח פרטי המצלמה

מקור וידאו SDK

מקור הווידאו SDK מספק את היכולת לשלב כמעט כל מקור וידאו חי או מוקלט (מצלמות IP, NVRs, DVRs, וכו ') לתוך מערכת ומאפשר ליצור שיטות לגלות, להציג, לנתח ולהקליט וידאו. זה גם מאפשר שילוב אינטגרציה עם יציאות והתקני I\O, כמוכן משיכת מידע על גילוי תנועה ממכשירי צד שלישי.

אחסון SDK

SDK לאחסון מאפשר למפתחים לקרוא או לכתוב לכל מיקום אחסון: מקומי, מרוחק, ואפילו לענן. יצירת תוסף אחסון דורשת יישום של פונקציות סטנדרטיות כגון: I/O Stream, אם הקובץ קיים, למחוק קובץ, רשימת קבצים בתיקיה, וכו'. ה- SDK לאחסון מכיל דוגמה לשימוש בשרת FTP כמיקום אחסון.
