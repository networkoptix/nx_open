服务器API

伺服器 API 提供一套标准的 API 供第三方厂商呼叫. 这让开发者 几乎可取用所有在标准 Client 上能操作的系统功能: 调阅/管理系统资源 (伺服器, 摄影机, 使用者, 权限), 从系统中拉取即时影像或录影资料, 生成事件, 建立规则, 控操 PTZ 摄影机以及更多.

API 文档

视频源集成 SDK

影像源整合 SDK 提供整合几乎任何即时影像或录影装置的能力 (IP摄影机, NVR, DVR等等.) SDK 可使得搜寻, 显示, 分析以及录存影像变得可能. 这也支援整合设备的 I/O 埠和移动侦测 资讯.

存储 SDK

储存装置 SDK 提供整合可能的储存装置进入系统. 这使得开发人员可 读取或写入任何储存装置︰本地, 远端或什至云端. 建立一个储存装置 外挂程式须可执行标准功能如：I/O串流, 档案是否存在, 删除档案, 列出资料夹所有档案等等. 储存装置 SDK 还包含如何使 FTP 伺服器成为储存装置的范例.
