通用事件生成器

通用事件是将事件从第三方系统发送到系统的方法， 可用于触发“事件和规则”引擎中的系统操作。 通用事件生成器是一种帮助构建HTTP泛型事件调用的工具。

通用事件生成器

服务器 API

伺服器 API 提供一套标准的 API 供第三方厂家呼叫. 让开发者几乎可取用所有在标准 Client 上能操作的系统功能: 调阅/管理系统资源 (伺服器, 摄像机, 使用者, 权限), 从系统中拉取实时流或录像资料, 生成事件, 建立规则, 操控 PTZ 摄像机以及更多.

摄像机详细信息面板

在 Web 管理界面中显示相机的其他开发人员友好信息：

  • 摄像头 ID
  • 所有可用视频流的链接
  • 从浏览器直接导出视频

要打开相机详细信息面板 - 到Web管理界面，选择查看 ， 选择目标相机，点击设置图标（齿轮），然后选择“相机详细信息”

开启摄像机详细信息面板

视频源集成 SDK

影像源整合 SDK 提供整合几乎任何实时流或录影装置的能力 (IP摄像机, NVR, DVR等等.) SDK 可使得搜寻, 显示, 分析以及录存影像变得可能. 这也支援整合设备的 I/O 端口和移动侦测资讯.

获取 SDK

存储 SDK

存储装置 SDK 提供整合存储装置进入系统. 这使得开发人员可读取或写入任何存储装置︰本地, 远端或云端. 建立一个存储装置插件须可执行标准功能如：I/O串流, 档案是否存在, 删除档案, 列出资料夹所有档案等等. 存储装置 SDK 还包含如何使 FTP 服务器成为存储装置的范例.
