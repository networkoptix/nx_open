通用事件生成器

通用事件是将事件从第三方系统发送到系统的方法， 可用于触发“事件和规则”引擎中的系统操作。 通用事件生成器是一个可帮助开发和整合人员的构建工具，             并可测试构建HTTP泛型事件。

通用事件生成器

服务器 API

REST API 能讓開發人員訪問幾乎所有系統功能， 客戶端：查詢/管理系統資源（服務器，攝像機，使用者，權限), 從系統中索取实时流或視頻資料, 生成事件，建立規則，操控PTZ攝像機以及更多功能。

API测试工具（新）             - API测试互动式文件和工具
            API变更日志             - 上次更新后的API重大更新列表
            API文档             - 可用服务器API的完整列表

摄像机详细信息面板

在 Web 管理介面的浏览标签页上显示对各个摄像机的其他友好开发人员讯息.

  • 摄像机 ID
  • 所有可用视频流的链接
  • 从浏览器直接导出视频

要打开相机详细信息面板 - 到Web管理界面，选择查看 ， 选择目标相机，点击设置图标（齿轮），然后选择“相机详细信息”

开启摄像机详细信息面板

视频源SDK

视频源SDK提供了能把几乎任何实时或录制的视频源（网路摄像机，             NVRs，DVRs等）整合到系统中的功能，并可用于发现、显示、分析             和录制视频。 还允许整合 I/O 端口和设备，并从第三方设备获取移动侦测 信息。

获取 SDK

存储 SDK

存储 SDK使得开发人员可读取或写入任何存储装置︰本地, 远端或云端. 创建存储插件需要执行标准功能，例如：I/O 流，如果文件 存在，删除文件，文件夹中的列表文件等。 存储 SDK 包含使用 FTP 服务器作为存储位置的示例。
