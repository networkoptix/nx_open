Generátor všeobecných událostí

Všeobecná událost je metoda odesílání událostí ze systémů třetích stran do systému, který může být použit k spuštění systémových akcí v nástroji Akce a pravidla. Generátor všeobecných událostíje nástroj, který vývojářům a integrátorům pomáhá vytvářet a testovat upozornšní všeobecných událostí HTTP.

Server API

REST API, které vývojářům poskytuje přístup k téměř všem funkcím systému dostupným v běžném klientovi: dotazování / správa systémových prostředků (servery, kamery, uživatelé, oprávnění), stahování živého nebo nahraného videa ze systému, generování událostí, vytváření pravidla, ovládání PTZ kamer a další.

Panel podrobností o kameře

Zobrazí další vývojářský důležité informace o kamerách ve webovém rozhraní administrátora:

  • Camera ID
  • Odkazy pro všechny dostupné video streamy
  • Přímý export videa z prohlížeče

Chcete-li otevřít panel Podrobnosti o kameře - přejděte do webového rozhraní administrátora, zvolte Zobrazit , vyberte požadovanou kameru, klikněte na ikonu nastavení (ozubené kolo) a poté zvolte "Detaily kamery"

SDK zdroje videa

SDK poskytuje možnost integrovat téměř jakýkoli živý nebo zaznamenaný video zdroj (IP kamery, NVR, DVR atd.) Do systému a umožňuje vytvářet metody pro objevování, zobrazování, analýzu a nahrávání videa. To také umožňuje integrovat I / O porty a zařízení, stejně jako informace o detekci pohybu údaje ze zařízení 3. stran.

SDK pro úložiště

Úložiště SDK umožňuje vývojářům číst nebo zapisovat do libovolného umístění úložiště: místního, vzdáleného a dokonce cloudového. Vytvoření zásuvného modulu úložiště vyžaduje implementaci standardních funkcí, jako například: I / O stream, pokud existuje soubor, smazat soubor, seznam souborů ve složce atd. Úložiště SDK obsahuje příklad použití serveru FTP jako umístění úložiště.
