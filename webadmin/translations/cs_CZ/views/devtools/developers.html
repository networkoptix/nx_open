Generátor všeobecných událostí

Všeobecná událost je metoda odesílání událostí ze systémů třetích stran do systému, který může být použit k spuštění systémových akcí v nástroji Akce a pravidla. Generátor všeobecných událostí je nástroj, který pomáhá vytvářet volání všeobecných HTTP událostí.

Server API

Rozhraní API serveru poskytuje standardní sadu volání API pro systémy třetích stran. To dává vývojářům možnost přístupu téměř ke každé funkci systému dostupné pro běžného klienta: zjistit / spravovat systémové prostředky (servery, kamery, uživatele, oprávnění), prohlížet živé nebo nahrané video systému, vytvářet události, vytvářet pravidla, ovládat PTZ kamery a další.

Panel podrobností o kameře

Zobrazí další vývojářský důležité informace o kamerách ve webovém rozhraní administrátora:

  • ID kamery
  • Odkazy pro všechny dostupné video streamy
  • Přímý export videa z prohlížeče

Chcete-li otevřít panel Podrobnosti o kameře - přejděte do webového rozhraní administrátora, zvolte Zobrazit , vyberte požadovanou kameru, klikněte na ikonu nastavení (ozubené kolo) a poté zvolte "Detaily kamery"

Sada SDK pro integraci zdroje videa

Sada SDK pro integraci zdroje videa umožňuje prakticky do systému integrovat jakýkoli zdroj živého videa nebo záznamu (IP kamery, NVR, DVR atd.). SDK umožňuje vytvářet metody pro vyhledávání, zobrazení, analýzu a záznam videa. Umožňuje také integraci I/O portů zařízení a informací o detekci pohybu.

SDK pro úložiště

SDK pro úložiště umožňuje integrovat potenciální úložiště do systému. Umožňuje vývojářům číst nebo zapisovat do jakéhokoli úložiště: lokální, vzdálené a nebo cloud. Vytvoření pluginu pro úložiště vyžaduje implementaci standardních funkcí, jako je například: I/O stream, pokud soubor existuje, odstranit soubor, seznam souborů ve složce atd. SDK pro úložiště obsahuje také příklad použití FTP serveru jako místa pro úložiště.
