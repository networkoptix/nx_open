Документация по серверу

Эти настройки доступны:

Для Windows в разделе реестра: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\"{{Config.productName}}"\"{{Config.productName}} Media Server".
Например: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Hanwha\Hanwha Media Server

Для всех других платформ он находится в файле конфигурации: /opt/{{Config.productName}}/mediaserver/etc/mediaserver.conf
Например: /opt/hanwha/mediaserver/etc/mediaserver.conf
