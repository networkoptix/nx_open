База знаний для разработчиков

Генератор произвольных событий

Произвольное событие - способ отправки событий из сторонних систем. Произвольные события могут использоваться для запуска системных действий в механизме событий и правил. Generic Event Generator - это инструмент, который помогает разработчикам и интеграторам создавать и тестировать HTTP-вызовы Generic Event.

API сервера

REST API дает возможность разработчикам получать доступ к почти каждой функции системы, доступной в обычном клиенте: запросу и управлению системными ресурсами (серверы, камеры, пользователи, полномочия), получению видео в реальном времени или архива из системы, генерирации событий, созданию правил, управлению PTZ-камерами и многому другому.

Инструмент тестирования API (новый) - интерактивная документация с инструментами для тестирования вызовов API
API-журнал изменений - список изменений в API, могущих затронуть текущий функционал
Документация API - полный список доступных вызовов API сервера

Панель сведений о камере

Отображает дополнительную информацию по отдельным камерам, полезную для разработчиков, во вкладке "Просмотр" в Web Admin:

  • Идентификатор камеры
  • Ссылки для всех доступных видеопотоков
  • Прямой экспорт видео из браузера

Чтобы открыть панель сведений о камерах - перейдите в интерфейс веб-администратора, выберите Вид, выберите желаемую камеру, щелкните значок настроек (шестеренка), затем выберите "Сведения о камере"

SDK источника видео

SDK источника видео предоставляет возможность интегрировать практически любой источник видео (IP камеры, NVR, DVRs и т.д.) в систему и позволяет создавать методы для обнаружения, отображения, анализа и записи видео. Он также позволяет интегрировать порты и устройства ввода/вывода, а также получать информацию об обнаружении движения от сторонних устройств.

SDK для интеграции хранилищ

SDK хранилища позволяет разработчикам выполнять операции чтения/записи в любое место хранения: местное, удаленное и даже облачное. Создание плагина хранения требует реализации стандартных функций, таких как: поток ввода-вывода, если файл существует, удаление файла, список файлов в папке и т.д. Storage SDK содержит пример использования сервера FTP в качестве хранилища.
