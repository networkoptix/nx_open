Генератор произвольных событий

A Generic Event is a method of sending events from 3rd party systems to a System, which can be used to trigger System actions in the Events & Rules engine. The Generic Event Generator is a tool which helps build HTTP Generic Event calls.

API сервера

API сервера обеспечивает стандартный набор API вызовов для внешних систем. Это дает возможность разработчикам получать доступ к почти каждой функции системы, доступной в обычном клиенте: запросу и управлению системными ресурсами (серверы, камеры, пользователи, полномочия), получению живого видео или архива из системы, генерирации событий, созданию правил, управлению PTZ-камерами и многому другому.

Панель сведений о камере

Displays additional developer-friendly information for cameras in the Web Admin interface:

  • Идентификатор камеры
  • Ссылки для всех доступных видеопотоков
  • Прямой экспорт видео из браузера

To open the Camera Details Panel - navigate to the Web Admin interface, choose View, select the target camera, click on the settings icon (gear), then choose "Camera Details"

SDK для интеграции источников видео

SDK предоставляет возможность интеграции практически любого источника живого или записанного видео (IP-камер, NVR, цифровых видеорегистраторов, и т. д.) в систему. SDK позволяет создавать методы для обнаружения, отображения, анализа и записи видео. Также возможно интегрировать порты устройств ввода-вывода и данные обнаружения движения.

SDK для интеграции хранилищ

SDK хранения данных предоставляет возможность интеграции потенциальных хранилищ в систему. Это позволяет разработчикам выполнять операции чтения/записи в любое место хранения: местное, удаленное и даже облачное. Создание плагина хранения данных требует реализации стандартных функций таких как: потоковый ввод-вывод, проверка существования файла, удаление файла, получения списка файлов в папке и др. SDK также содержит пример использования FTP-сервера в качестве хранилища.
