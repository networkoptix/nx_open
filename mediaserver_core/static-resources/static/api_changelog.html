A number of Rest API methods and params have been renamed in 3.0.0, breaking backwards compatibility (except where deprecated names are supported).

Breaking compatibility

Renamed methods: 
    /ec2/getServerUserAttributes -> /ec2/getMediaServerUserAttributesList
    /ec2/saveServerUserAttributes -> /ec2/saveMediaServerUserAttributes
    /ec2/saveServerUserAttributesList -> /ec2/saveMediaServerUserAttributesList
    /ec2/getCameraUserAttributes -> /ec2/getCameraUserAttributesList
    [proprietary] /ec2/getClientInfos -> /ec2/getClientInfoList
    /api/businessEvents -> /api/getEvents
    /ec2/getBusinessRules -> /ec2/getEventRules
    [proprietary] /ec2/saveBusinessRule -> /ec2/saveEventRule
    [proprietary] /ec2/removeBusinessRule -> /ec2/removeEventRule
    [proprietary] /ec2/resetBusinessRules -> /ec2/resetEventRules
    [proprietary] /ec2/broadcastBusinessAction -> /ec2/broadcastAction
    [proprietary] /ec2/execBusinessAction -> /ec2/execAction
Changed reply format XML -> JSON in methods: 
    /api/manualCamera/search
    /api/manualCamera/status
    /api/manualCamera/stop
    /api/statistics
    /api/storageSpace
    /api/storageStatus

Retaining compatibility until expired

Renamed param cameraID -> cameraId (deprecated name support expires by 3.1) in methods: 
    /ec2/getCamerasEx
    /ec2/getCameraUserAttributesList
    /ec2/saveCameraUserAttributesList
    /ec2/saveCameraUserAttributes
Renamed param serverID -> serverId (deprecated name support expires by 3.1) in methods: 
    /ec2/getMediaServerUserAttributesList
    /ec2/saveMediaServerUserAttributesList
    /ec2/saveMediaServerUserAttributes
Renamed param phisicalMemory -> physicalMemory (deprecated name support expires by 3.1) in methods: 
    /ec2/saveClientInfo
    /ec2/getClientInfoList
Renamed param preferedServerId -> preferredServerId (deprecated name support expires by 3.1) in methods: 
    /ec2/saveCameraUserAttributesList
    /ec2/saveCameraUserAttributes
    /ec2/getCameraUserAttributesList
    /ec2/getCamerasEx
Renamed param res_id -> cameraId (deprecated name support expires by 3.1) in methods: 
    /api/doCameraDiagnosticsStep
    /api/setCameraParam
    /api/getCameraParam
Renamed param resourceId -> cameraId (deprecated name support expires by 3.1) in methods: 
    /api/ptz
Renamed and merged together params mac + physicalId -> cameraId (deprecated names support expires by 3.1) in methods: 
    /ec2/bookmarks
Renamed param physicalId -> cameraId (deprecated name support expires by 3.1) in methods: 
    /api/image
    /api/getEvents
    /ec2/recordedTimePeriods
Changed GET -> POST (deprecated method support expires by 3.1, check extra details in documentation) in methods: 
    /api/manualCamera/add
Changed RTSP Streaming URL (deprecated URL support expires by 3.1): 
    /<physicalId> -> /<cameraId>
Changed HTTP Streaming URL (deprecated URL support expires by 3.1): 
    /media/<physicalId>.<format> -> /media/<cameraId>.<format>
Changed HLS Streaming/Download URL (deprecated URL support expires by 3.1): 
    /hls/<physicalid>.<format> -> /hls/<cameraId>.<format>
