Detailed Description

template<typename T>
class Argus::Array2D< T >

A template class to hold a 2-dimensional array of data.

Data in this array is tightly packed in a 1-dimensional vector in row-major order; that is, the vector index for any value given its 2-dimensional Location is index = location.x + (location.y * size.x); Indexing operators using iterators, 1-dimensional, or 2-dimensional coordinates are provided.
